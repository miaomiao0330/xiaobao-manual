# 录播课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/eftlt93vcp4t8hel

场景释义

**内容即资产，放大名师价值**：将优秀教师的精华课程沉淀为标准化的高品质录播课，一次录制，无限复用。这不仅突破了名师的时间与空间限制，更将其个人教学魅力转化为机构的可传承、可放大的核心数字资产

**灵活即服务，完善教学闭环**：录播课程可以一键分享给家长，方便学员复习巩固，也让家长直观了解教学品质；同时支持关联补课任务，学员未到课，机构无需补课时，可以选择推送对应录播课，确保每位学员的学习进度

**开放即获客，观看即线索**：支持将录播课设为公开课，吸引潜在家长在家校端自由观看。巧妙设置观看门槛，未购课家长需授权手机号，自然地将内容吸引力转化为精准销售线索

操作步骤

创建总部录播课

**路径**：总部管理后台 >> 跨校办理 >> 录播课 >> 总部设置 >> 新建录播课 >> 填写录播课名称、详情介绍等基础信息 >>设置上下架状态>> 根据机构实际需求，选择录播课可见范围 >> 根据机构实际需求，设置录播课属性，例如科目等

注：1. 上架录播课，才能让家长观看

2. 如果将录播课设置为公开状态，任何人都可以观看；非机构学员观看前，需要授权手机号，因此设置为公开课，可以为机构带来线索

3. 录播课的文件仅支持通过云盘上传，云盘操作手册可[点此查看](https://www.yuque.com/schoolpal-wiki/manual/nafy610d2t7t7an9#oknvb)

给分校分发课程

**路径**：创建录播课时，直接点击「保存并分发」；或者在录播课裂变，点击「分发」>> 选择分发校区，设置录播课总部是否持续同步分校，是否支持分校自主编辑

注：如果总部创建录播课时，设置了可见学员范围、录播课属性，当分校没有对应学员报读课程或课程属性时，将同步失败，各类失败原因和修改方式如下：

- 分校无录播课关联的科目：前往【总部管理后台 >> 教务设置 >> 课程设置 >> 校区设置】，开启【同步总部课程设置】按钮
- 分校无录播课关联的课程属性：前往【总部管理后台 >> 教务设置 >> 课程设置 >> 校区设置】，开启【同步总部课程设置】按钮
- 分校无录播课关联的课程类别：前往【总部管理后台 >> 教务设置 >> 课程类别 >> 校区设置】，开启【同步总部课程类别】按钮
- 分校无录播课适用课程：修改录播课的适用课程，或者总部将适用课程分发给分校
- 分校无录播课适用的学员属性：修改录播课的适用学员属性，或者总部将学员属性同步分校
- 分校无录播课适用的渠道：修改录播课的适用渠道，或者总部将渠道设置同步分校

录播课设置

**路径**：总部管理后台 >> 跨校办理 >> 录播课 >> 校区设置 >> 设置「是否允许校区自建录播课」![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1768188060122-cbed0817-57c0-4ea9-85e6-20b661e85c16.png)

录播课上下架、删除

路径：总部管理后台 >> 跨校办理 >> 录播课 >> 总部设置 >> 筛选上下架状态 >> 批量操作「上架」、「下架」、「删除」

注：录播课删除后，当家长通过家校端、分享链接观看时，将显示录播课已下架；正在播放过程中的录播课不受影响
