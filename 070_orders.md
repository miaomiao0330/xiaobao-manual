# 订单管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/orders

一、场景释义

记录学校导入/报名操作的所有订单，方便老师对学校的订单进行统一的查看和导出。

二、操作步骤

App 端

（一）作废订单

**不同类型订单支持废除**

- 如果订单课程均未结课，并且未关联转课、退费订单，支持作废，已报课时数清空；
- 转课订单支持废除，转入课时会返还原学费账户；
- 退费订单支持废除，已退学费返还至原学费账户；
- 包含结课课程的订单不支持废除，如果点击废除，可以跳转查看报读课程详情；
- 关联转课、退费订单的课程订单不支持废除，可以查看关联订单详情；
- 课程订单关联收银宝账单，如果课程订单作废后，之前关联的收银宝账单支持重新关联到正确的课程订单下
- 报课订单作废默认选中“作废后不结课”，如果选择“作废后结课”在订单作废后，查询订单下所有课程的学费账户是否为 0，如果有为 0 则将为 0 的学费账户的课程结课；如果选择“作废后不结课”，则保持当前作废订单逻辑不做修改

路径：工作台 >> 订单 >> 订单详情>> 点击“废除订单” >>确认

（四）订单相关操作

补费

直接补费

**路径：**财务中心>>订单管理>>订单列表>>选择欠费订单>>补费>>直接补费

**注：**若补费后订单仍存在欠费，则会显示【分摊补费】按钮，可自定义设置每项办理内容的分摊补费金额

![image.png](https://cdn.nlark.com/yuque/0/2024/png/12534096/1728480135029-7e4d8534-4e21-4197-89c6-807c2ad28e69.png)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762173089201-b439554c-7edd-4fa3-a8ca-f8c984056426.png)直接补费

调整优惠后补费

**路径：**财务中心>>订单管理>>订单列表>>选择欠费订单>>补费>>调整优惠后补费

**注：**选择【调整优惠后补费】，选择后系统会将原订单作废，并根据调整后的整单优惠、使用储值账户充值金额、使用储值账号赠送金额、使用优惠券等优惠信息自动创建新订单

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762172923787-b125a2d0-1472-4a15-8c7c-9e28959bcaf8.png)调整优惠后补费![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762172904660-29e2e80a-47d6-4c9d-8b49-61e24be331fe.png)调整优惠

设为坏账

**路径：**财务中心>>订单管理>>订单列表>>选择欠费订单>>设为坏账/取消坏账

**注：**若看不到操作按钮，请检查角色是否有该权限点，默认分配给校区管理员、财务、销售

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762172793753-94de26bb-34b1-4dd6-8416-7a4662cbc51b.png)

教学用品发货

**路径：**财务中心>>订单管理>>查看订单详情>>发货

![image.png](https://cdn.nlark.com/yuque/0/2023/png/26909528/1691220368549-cf1e178e-0b90-4290-ab2b-03206f2f46cf.png)教学用品发货

收据下载或打印

**路径：**财务中心>>订单管理>>选择订单>>下载/打印收据/发送短信，或在订单详情中查看操作

注：收据支持显示校区名称或不显示，可以在 App-业务设置 中开关。

**路径：**财务中心>>订单管理>>选择订单>>废除订单
