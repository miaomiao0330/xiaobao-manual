# 补课管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/makeuplesson

一、场景释义

**安排插班补课**：当不需要为请假学员单独安排时间补课，而是在已有排课中安排时，使用“插班补课”进行操作即可。

**添加补课记录**：当需要为请假学员单独安排时间补课时，使用“添加补课记录”进行操作即可。

**无需补课**：若出于某种原因，不需要为学员另行安排补课时，可使用“无需补课”。需注意，操作后系统将自动删除此条补课记录，后续也无法为其安排补课。

三、常见问题

补课管理可以搜索需要补课学员吗？

可以的

补课管理中没有显示按时段和按金额扣费请假学员？

需要去**APP首页 >> 我的 >> 业务设置 >> 课程与课消设置 >> 收费方式 >> 按金额/时段开启“学员补课”按钮**，允许两种收费方式下的学员补课。

注：只针对规则设置后请假的学员生效。之前学生需要补课，请先使用未排课点名，或者添加临时学员方式补记。

如果补课学员未到课，可以再次安排补课吗？

如果课程点名前，学员未来，可以去补课管理中给学员取消补课，重新安排插班补课或等学员来了在点名记录中，重新添加补课学员。
