# 收银宝（总部）

> 来源: https://www.yuque.com/schoolpal-wiki/manual/chi3yn9gkhbh6a1z

一、场景释义

总部管理在授权对接设置中授权收银宝账号，对支付宝、花呗、微信等多种账号进行开通认证等操作，提升同步管理效率。

二、操作步骤

收银宝

授权收银宝账户

路径：进入总部管理后台 >> 设置统管 >> 授权对接设置 >> 收银宝 >> 授权账户 >> 输入手机号、验证码后完成

注：仅超管可以操作，机构自行给有需要的角色勾选权限

![image.png](https://cdn.nlark.com/yuque/0/2025/png/12534096/1754978865338-c2a62559-ed45-4fdf-9f8d-8d948e0fc159.png)![image.png](https://cdn.nlark.com/yuque/0/2025/png/12534096/1754978879713-5c006174-78ab-4c40-913f-85b608607908.png)

解除收银宝账户

路径：进入总部管理后台 >> 设置统管 >> 授权对接设置 >> 收银宝 >> 解除授权 >> 确认解除授权
