# TQ 企友对接

> 来源: https://www.yuque.com/schoolpal-wiki/manual/ai

一、场景释义

接入企友科技（TQ外呼）外呼服务后，老师可在系统内直接拨号联系学员，通话记录和号码自动沉淀至「意向学员」，高效管理线索。

二、操作步骤

（一）授权外呼账号

机构已自行购买企友科技外呼服务后，请找企友科技索取 ID 和 Secret ，在总部后台/授权中授权即可完成绑定。

**多校区机构操作路径：**总部管理后台 >> 设置统管 >> 授权对接设置 >> 外呼对接 >> 输入 Secret 和 ID >> 完成绑定

三、常见问题

本机外呼也有录音吗？也会有通话记录转文字、本次通话分析吗？

没有，只有通过企友科技（TQ外呼）拨打的电话才会有录音以及分析。通过本机呼出的内容受到隐私保护，内容不会上传至系统。

通过企友科技（TQ外呼）呼出的电话，会同步到校宝App 的意向学员吗？

不会同步。目前仅支持授权绑定企友科技（TQ外呼）的机构在校宝App 呼出并生成通话记录。

手机已经安装了企友科技，无法呼出，是有什么手机型号、版本号的限制吗？

目前 TQ 还没有支持所有的手机型号呼出，详情查看下表：

| **品牌系列** | **适配系统版本详情** |
| --- | --- |
| 小米、红米 | 1.MIUI 14<br><br>2.澎湃OS 2.0 |
| 华为、荣耀 | 1.MagicOS 9.0<br><br>2.HarmonyOS 5.0以下版本（兼容Android系统环境） |
| OPPO、realme | 1.ColorOS 15<br><br>2.realme UI 7.0 |
| vivo | OriginOS 3.0 |
| 魅族（meizu） | Flyme 9.0 |
| 摩托罗拉（motorola） | MYUI 6.0（基于Android系统开发，兼容Android相关生态） |

外呼后，在校宝App 的智能外呼上没有找到录音或录音显示正在上传？

先到 TQ 后台检查音频是否存在：

1. 如果 TQ 后台也没有记录或显示待上传，说明是 TQ 平台问题，联系 TQ 排查
2. 如果 TQ 后台有录音记录，大概率是回调问题（例如超时失败），联系校宝排查
