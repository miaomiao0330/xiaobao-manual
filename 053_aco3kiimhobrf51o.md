# 班级错题本 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/aco3kiimhobrf51o

一、场景释义

老师通过课后任务或拍照试卷为班级学员统一收录错题后，即可在校宝 App PC 端快速定位对应班级，查看班级整体错题分布、各学员错题明细及高频错题排行，支持按班级进行错题重练，还可基于班级共性错题一键启用 AI 举一反三，智能生成适配全班训练需求的相似题，实现班级层面的精准巩固与集体提效。

二、操作步骤

PC 端

（一）错题收录

![](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772454079176-686a8f00-7606-470c-a993-8844fb2566d1.png?x-oss-process=image%2Fquality%2Cq_10)[![](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772454079176-686a8f00-7606-470c-a993-8844fb2566d1.png?x-oss-process=image%2Fquality%2Cq_10)AI 错题本老师在线批改作业时标注错题或使用校宝App 拍照上传错题，系统自动加入到该学员的错题本中，并实时同步到家长端校宝App 操作手册](https://www.yuque.com/schoolpal-wiki/manual/bh9iwg26vr09x6i6)

（二）管理班级错题

**路径：**教务中心 >> 班级管理 >> 选中班级 >> 点击错题本 >> 可选择学科学段、筛选题型以及其他筛选条件 >> 可对查看错题统计数据，包括答题人次、答错人次、错误率，以及答错学员明细

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1780319095534-7fb9922e-6aaf-4bff-9711-c4c7d8267787.png)

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1780319146515-4377559e-0696-4513-ab48-6a7b07c3f06b.png)错题统计

（三）薄弱知识点

**路径：**教务中心 >> 班级管理 >> 选中班级 >> 点击薄弱知识点 >> 可选择学科学段、做题时间 >> 数据总览、可查看班级下全部学员的知识点分析图谱、难易度占比分析、错题学员明细

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779963366745-19a83654-0cdf-4671-9f90-e2ad4f053b71.png)

错题重练

**路径：**教务中心 >> 班级管理 >> 选中班级 >> 点击薄弱知识点 >> 选中并点击知识点 >> 错题重练 >> 选择错题加入试题篮 >> 组卷

注：系统会自动根据知识点筛选相关错题

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1780319695825-0a5f952c-7a3e-4191-a9d2-e60ae719c106.png)

选择错题

可直接将班级学员做错的对应知识点的错题加入到试题篮中

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1780319732922-1e211109-ab62-4e63-9107-a30cfe5fd7d4.png)

完成选题并组卷

点击完成选题按钮，进入组卷页面，输入试卷名称、学科学段等基础信息，并选择【是否需要添加个性题】

注：开启后，在下载试卷或布置作业时，会从学员错题本中自动选择对应试题加入到试卷中

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779961339668-f967ceed-6238-4373-9aad-19f6bbb24eae.png)

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779961965338-798dd128-4329-4be8-a2da-5303d3e0a058.png)

1.3. 下载试卷

完成组卷后，即可去下载试卷，下载时，可选择仅通用题、仅个性题或通用题+个性题三种试卷形式

注：选择多名学员时，个性题将自动从学员错题本中选取后插入到试卷中，也可手动为每个学员从题库或者错题本中选择个性题

AI 举一反三
