# 招生宝

> 来源: https://www.yuque.com/schoolpal-wiki/manual/xklenfprhbfvd7t1

一、场景释义

线上招生引流工具，系统自带拼团、砍价、分销等招生活动，学员报名参加还可以一键分享朋友圈，帮助机构进行招生活动并扩大品牌宣传。

二、操作步骤

App 端

**路径：**工作台 >> 招生宝 >> 跳转“校宝招生宝”小程序 >> 切换顶部标签选择需要创建的模板类型 >> 点击模板并点击“一键领取方案”>> 编辑活动内容 >> 同意用户协议并发布活动

三、常见问题

招生宝活动中，家长交的钱都收到哪里了？

在“校宝钱包”里。

**路径：**招生宝小程序>>底部栏“我的”>>点击“我要提现”>>点击“去提现”（首次需绑定银行卡）

为什么校宝钱包里收到的钱比实际家长付款的要少？

微活动功能基于微信端，微信会收取手续费。

发布活动后还能编辑嘛？

可以的，发布后，涉及价格、数量等相关设置无法再编辑，活动的文字及图片内容都可以编辑。

**路径：**招生宝小程序>>底部栏“管理”>>找到对应活动并点击右下角“…”>>点击“编辑活动”

可以把活动链接分享到朋友圈吗？

可以的，小程序会生成带有活动二维码的海报，老师可以保存后发到朋友圈。小程序链接本身无法直接转发到朋友圈，但可以转发到聊天框。

**路径：**招生宝小程序>>底部栏“管理”>>找到对应活动并点击右下角“分享”>>点击“保存活动海报”

通过活动报名的学员怎么查看？

**路径：**校宝App>>工作台>>招生中心>>线索管理
