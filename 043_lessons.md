# 课程商品

> 来源: https://www.yuque.com/schoolpal-wiki/manual/lessons

场景释义

- 机构专门创建特价体验课，用于试听、引流，学员报名后仍属于「意向学员」
- 机构可创建常规课、通用课等不同类型的课程模式，学员报名后即为「正式学员」；机构也可同时在线管理教学用品，支持学员在线或线下购买
- 机构在创建课程的时候，可将正式课当作试听课来运营，提升机构教务运营自由度

操作步骤

App 端

课程商品

创建课程

**路径**：课程商品 >> 创建商品 >> 填写课程信息 >> 报价单 >> 设置课程价格 >> 开启微校售卖，或者开启体验价

注：1. 一门课程可设置多个报价单，可选择「单个」或「全部」价格类型开启微校售卖

2. 未报课学员购买体验价的商品，仍是意向学员状态

3. 如需创建通用课程，则需要先打开通用课程开关：我的 >> 业务设置 >> 课程设置 >> 打开「通用课」

**名词解释：****班级授课**：此课程对应的授课模式，只要不是一对一进行授课，就都选择班级授课
**1 对 1 授课**：只适用于一个老师针对一个学员单独上课的授课模式
**通用课**：例如初三数学是初三数学小班课和初三数学大班课的通用课，那买了初三数学课，就可以同时上初三数学的小班课和大班课；此规则适用于班课、1 对 1 课程
**按课时收费**：按「购买课时数」定价，以课时为单位计费
**按时段收费**：按「天 / 自然月 / 自然年」定价，以「天」为单位计费
**按金额收费**：给每节课设置一个价格，每上一节课，就扣除对应学费
**报价单名称**：此种收费标准对应的名称，比如「小班课价格」、「月卡」、「年卡」等，便于学员报名时快速选择对应价格

微校商城售卖

2.1 课程购买设置

**路径：**课程商品 >> 编辑课程 >> 微校商品详情 >> 微校课程设置 >> 微校展示、购买限制 >> 确定 >> 保存

注：微校展示开关，不影响机构将课程分享给学员/家长
购买限制规则：
允许老生购买的规则：如果选择任意课程的在读学员，则机构所有在读学员均可购买此课程；如果选择 A、B 课程的在读学员和历史学员，则机构只有购买过 A、B 课程的学员可购买此课程
允许新生购买的规则：如果选择仅新生购买，则只有意向学员可以购买此课程
限购 1 单规则：如果开启，则学员仅能在线上购买一次、单笔，即使此课程有多个报价单，也只能选择其中一个报价单进行购买

2.3 家长在线购买

**路径**：家长进入「校宝家校通」小程序 >> 点击课程商品 >> 在线选择商品、班级（如机构设置）>> 在线付款，学员在校宝系统里自动完成报课、选班，课程订单和付款账单自动完成对账

注：1. 机构需要开通收银宝，认证线上支付通道，家长才可以在线购买。收银宝免费开通方式：[收款账户设置](https://www.yuque.com/schoolpal-wiki/manual/gpsvc5alshcpfz62#JP1hz)

2. 家长已在线选择课程，但还未付款，订单会在规定时间内默认取消。设置方式：我的 >> 业务设置 >> 财务设置 >> 交易规则设置 >> 微校未支付订单过期时间

购买教学用品

4.1 线下购买

**路径：**报名续费 >> 选择学员 >> 办理内容选择「按课程商品」 >> 选择教学用品，设置价格、数量和优惠 >> 点击结算，确认订单并支付

注：1. 未开启自动发货：购买教材付完款后，会提示是否发货，如果点击发货，订单完成；如果点击不全部发货，可以在订单管理中再选择发货

2. 开启自动发货：订单完成后，直接发货成功。自动发货开启路径：[点此查看](#UmS4b)

创建校区课程

2.1 手动创建课程

**路径**：课程商品 >> 创建商品 >> 填写课程信息 >> 报价单 >> 设置课程价格 >> 开启微校售卖，或者开启体验价

注：1. 一门课程可设置多个报价单，可选择「单个」或「全部」价格类型开启微校售卖

2. 未报课学员购买体验价的商品，仍是意向学员状态

3. 如需创建通用课程，则需要先打开通用课程开关：我的 >> 业务设置 >> 课程设置 >> 打开「通用课」

**名词解释：****班级授课**：此课程对应的授课模式，只要不是一对一进行授课，就都选择班级授课
**1 对 1 授课**：只适用于一个老师针对一个学员单独上课的授课模式
**通用课**：例如初三数学是初三数学小班课和初三数学大班课的通用课，那买了初三数学课，就可以同时上初三数学的小班课和大班课
**按课时收费**：按「购买课时数」定价，以课时为单位计费
**按时段收费**：按「天 / 自然月 / 自然年」定价，以「天」为单位计费
**按金额收费**：给每节课设置一个价格，每上一节课，就扣除对应学费
**报价单名称**：此种收费标准对应的名称，比如「小班课价格」、「月卡」、「年卡」等，便于学员报名时快速选择对应价格

三、常见问题

只能创建付费的体验课吗？

（1）单独创建体验课时，不支持设定 0 元的价格，但支持学员不付费参与体验课。

（2）可在给试听课排课时，直接添加意向学员，学员就可以不付费来上体验课，操作流程如下：课表 >> 点击右下角“+” >> 选择“试听课节” >> 填写体验课的排课信息，并选择“试听学员” >> 确认无误点击完成

报读通用课程的学员怎么分班

无需创建专门的班级给报读通用课程的学员，只需要将学员分配到对应课程的原有班级即可，支持一次性给学员分多个班级。

**路径**：学员档案 >> 找到对应学员 >> 点击“去分班” >> 选择要分入班级 >> 保存

![image.png](https://cdn.nlark.com/yuque/0/2021/png/326547/1628757845213-f1914eb2-f3b2-4f42-93aa-d16166357541.png)![image.png](https://cdn.nlark.com/yuque/0/2021/png/326547/1628757858421-afc71282-361c-43ff-9b12-3e5317dab9f9.png)

学员通用课程的课消怎么算

- 按照通用课程设置的价格来，与原有普通课程的价格无关
- 如学员报读了某普通课程的情况下，又报读了一门通用课程，且通用课程也包含这门普通课程。课消时会优先选择先报读的课程价格。

教学用品订单怎么筛选

在订单列表筛选「待处理」订单，通过查看办理内容确定教学用品，进入详情选择发货即可。

为什么有的教学用品只能选择停售或售卖，不能编辑

机构打开出入库管理前，创建的教学用品，在打开出入库管理后不支持编辑，只能选择售卖或停售；之后创建的教学用品可以查看详情、编辑等操作。

教学用品如何线上购买

- 需要机构在创建或编辑教学用品时，选择报价单对应上架微校，就可以实现家长在线选择、购买。
- 支持机构配置微校商品展示详情，包括图片、文字介绍，方便家长在线查看详情。

如果选择新规格的货品入库，教学用品对应可以修改吗？

可以的，比如机构在出入库管理中，添加了新的画笔规格，72 色油画棒，教学用品列表关联的画笔会有规格新增提示，对应编辑修改即可。

支持批量修改课程报价单吗？

支持的，首先登录到 PC 端管理后台：第一步，导出课程商品报价单；第二步，更新报价单相关字段；第三步，按要求填写模版的表格内容；

课程删除后可以创建同名课程吗？

可以，如果删除课程 A，机构可以再创建一个课程 A，如果已删除的课程 A 还原，则名称自动更改为 A（1）
