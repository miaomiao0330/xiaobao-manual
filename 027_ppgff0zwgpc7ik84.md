# 学员教材学杂费管理 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/ppgff0zwgpc7ik84

一、场景释义

学员购买教材、学杂费后，需要和机构退费，机构可以直接找到对应学员，发起退费操作，自主选择退费方式。

二、操作步骤

App 端

（一）学杂费退费

不欠费订单退费

**路径**：找到需要退费学员 >> 点击教材学杂，选择需要退费的学杂费 >> 选择退款方式，点击完成并提交

注：1. 如机构开启了收银宝收款，退费时选择收银宝，退费款项即可原路退到家长账户（[点此查看收银宝开通方式](https://www.yuque.com/schoolpal-wiki/manual/odxfrv)）

2. 拥有「学杂教材」、「退学杂教材费」权限的老师才能进行退费操作，校区管理员、教务、顾问默认拥有此权限

欠费订单退费

**路径**：找到需要退费学员 >> 点击教材学杂，选择需要退费的学杂费 >> 选择退款方式，点击完成并提交

注：1. 在退费时，将先补齐欠费金额，再呈现最终退费金额，因此当退费金额 ≥ 订单欠款，应退金额 = 退课金额-订单欠款，例如购买了多份保险，总计 150 元，实付 140 元，退费 100 元，最终的退费金额为 90 元；当退费金额 ＜ 订单欠款，则应退金额为 0，例如购买了多份保险，实付 10 元，欠费 140 元，退费 100 元，最终的退费金额为 0 元，欠费金额会变少

2. 实退金额 ≥0，若超出「应退金额」，超出部分计为「亏损费」；实退金额 ≥0，若小于「应退金额」，少于部分计为「手续费」

（二）教材退费

不欠费订单退费

**路径**：找到需要退费学员 >> 点击教材学杂，选择需要退费的教材 >> 分别填写退费、入库的教材数量 >> 选择退款方式，点击完成并提交

注：1. 如机构开启了收银宝收款，退费时选择收银宝，退费款项即可原路退到家长账户（[点此查看收银宝开通方式](https://www.yuque.com/schoolpal-wiki/manual/odxfrv)）2. 如机构开启「出入库管理」，则需要填写入库数量；如退费教材有破损，不宜入库，入库数量可为 03. 拥有「学杂教材」、「退学杂教材费」权限的老师才能进行退费操作，校区管理员、教务、顾问默认拥有此权限

欠费订单退费

**路径**：找到需要退费学员 >> 点击教材学杂，选择需要退费的教材 >> 选择退款方式，点击完成并提交

注：1. 退费时，将先补齐欠费金额，再呈现最终退费金额，因此当退费金额 ≥ 订单欠款，应退金额 = 退课金额-订单欠款；当退费金额 ＜ 订单欠款，则应退金额为 0，欠费金额会变少；例如购买了多份护膝，共计 60 元，实付 10 元，退费 60 元，最终的退费金额为 10 元

2. 实退金额 ≥0，若超出「应退金额」，超出部分计为「亏损费」；实退金额 ≥0，若小于「应退金额」，少于部分计为「手续费」

**路径**：点击查看收据 >> 选择打印收据样式，进行打印

（五）作废退费订单

**路径**：查看需要作废的学杂教材购买订单 >> 点击「废除订单」

注：1. 同时购买了课程和学杂教材，该课程进行了转课 / 退课 ，学杂教材发生过退费，则该课程购买订单无法废除；如需废除购买订单，需要在「订单记录」中，废除已发生过转课 / 退课 / 退费的订单

2. 单独购买了学杂教材，学杂教材发生过退费，购买订单无法废除；如需废除购买订单，需要在「订单记录」中，废除退费订单

3. 学杂教材在退单时，将费用退回到学员储值账户；在「订单记录」下操作作废时，作废金额高于学员储值账户金额，则无法废除成功
