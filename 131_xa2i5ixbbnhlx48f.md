# 跨校转课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/xa2i5ixbbnhlx48f

一、场景释义

多校区机构中存在跨校分班上课的业务，跨校转课通过跨校学员转移、跨校报名学费账户处理、跨校订单创建、跨校流水 / 报表处理，从而实现学员转校上课的功能，满足机构多校区的业务。

二、操作步骤

App 端

（一）设置总校管理权限

**路径：**总部管理 >> 员工管理 >> 选择任意一个员工 >> 开启设置为总部管理员 >> 完成

注：拥有总部管理员、跨校办理员权限的老师默认支持跨校业务办理

（二）跨校区转课

**路径：**总部管理 >> 跨校转课 >> 办理跨校转课 >> 填写转出校区、学员、课程 >> 填写转入校区、课程 >> 选择学员档案（如有）>> 确认并提交

**注：**1. 当学员在校区 B 无档案，从校区 A 转入校区 B 时，系统将在校区 B 中默认创建档案2. 当学员在校区 B 有同个联系方式和同姓名的档案，从校区 A 转入校区 B 时，系统将展示该联系方式下所有的档案供老师选择3. 当学员在校区 B 有同个联系方式，但没有同姓名的档案，转入校区 B 时，系统将展示所有该联系方式下的档案，且支持老师使用转出校区的学员档案

三、常见问题

跨校区转课的学员，以前的上课记录和点评记录会转到新校区吗？

学员在转课之前校区的上课记录、课堂点评等信息都仍会存在于原来校区，不会转移到新校区。

如果校区 A 有学员转到校区 B，学员在校区 B 消课，会影响校区 A 销售老师的业绩吗？

不会影响校区 A 销售老师的业绩。业绩和报课订单归属有关，报课订单不会随着转课而转移。

学员从校区 A 转到校区 B，需要重新关注家校平台吗？原来的家校数据还能查看吗？

学员家长需要重新在校区 B 绑定自己的孩子，家长可以通过切换校区的方式查看之前校区的家校数据。
