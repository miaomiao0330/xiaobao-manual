# 员工管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/afgvurok2o96542m

一、场景释义

可为需要登录系统的老师创建员工账号、更改权限、登记员工信息等，对校区内的员工进行规范化管理。

三、常见问题

非大陆手机号的员工可以登录系统吗？

可以，详情查看操作[新手任务](https://www.yuque.com/schoolpal-wiki/manual/wrs3s8#rzefv)

员工可以删除嘛？

员工不支持删除，可以将员工状态改为“离职”，在员工列表中默认隐藏“离职”状态的员工。

员工手机号可以更换嘛？

可以更换，分两种情况：

（1）员工已经用原手机号登录系统，激活过账号，现想修改为新手机号。

这种情况，只能老师自己登录系统，去我的 >> 点击姓名 >> 账号与安全 >> 更换手机号。

（2）员工未登录过系统，账号未激活。

这种情况，超管/校区管理员老师可以去员工管理 >> 员工信息 >> 修改员工手机号。

**注意：如果新手机号已在系统中激活过或者有云端的记录，那么更换需要联系实施经理走审批更换哦。**
