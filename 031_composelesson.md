# 一班多课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/composelesson

一、场景释义

本次新增一班多课功能。开启一班多课，机构可以设置组合课程，创建班级时关联组合课程，实现报名不同课程的学员在同一个班级中上课。

**路径：**App 业务设置 >> 课程与课消设置 >> 一班多课

三、常见问题

1.组合课程适用场景

小丽报名了舞蹈课 A ，小红报名了舞蹈课 B ，可以创建一个组合课程“舞蹈基础课”，将舞蹈课 A 和舞蹈课 B 加入这个组合课程，然后创建班级时，选择这个组合课程“舞蹈基础课”，即可实现报名不同课程的小丽和小红在一个班级上课。

2.组合课程模式中，点名时需要单独设置学员点名课程吗？

不需要点名时，上课老师只需对班级学员进行统一点名即可，系统会根据组合课程中多个课程的课消顺序对学员进行课消。
