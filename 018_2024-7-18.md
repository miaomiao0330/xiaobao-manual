# 报名续费

> 来源: https://www.yuque.com/schoolpal-wiki/manual/2024.7.18

一、场景释义

新学员 / 老学员来学校报名 / 续费，学校给学员在系统录入购买课程。

二、操作步骤

App 端

（一）创建/选择学员

**路径：**工作台 >> 报名续费 >> 点击【请选择学员】 >> 选择学员或新建 >> 填写学员信息 >> 确认无误点击【创建并选择】

![lQDPJxOYnthrZwzNCWDNBDiwIMqZvK8jAsgDjYuD1IBxAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670468948707-95e0acf8-b317-4153-b36b-94cb178ac230.jpeg)报名\续费![lQDPJyA0asy6NwzNCWDNBDiw-dKEXaQSfMoDjYuDawBxAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670469115923-3e39bf7a-3e72-4fe5-935a-7a2552a25ed2.jpeg)学员列表![lQDPJxjZw2v-7CzNCWDNBDiwTp2gtLzykbQDjY0zHkCFAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670469389000-134a5e65-fd82-4f6d-95c6-305a0380a406.jpeg)选择学员![lQDPJxPPMOQneYzNCWDNBDiwgtcMAbt7BJgDjYuCfgBxAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670469135765-5a79d385-a565-4f30-a2a5-c54ad8c98843.jpeg)创建新学员

（二）选择要报读的课程

**路径：**点击【请选择课程】 >> 选择课程 >> 选择对应的报价单 >> 填写购买的课时数或时段 >> 设置课程优惠 >> 点击完成

![image.png](https://cdn.nlark.com/yuque/0/2025/png/12534096/1764596319830-be763be4-b72d-4058-b1f1-24d215161881.png)选择学员![IMG_0077.PNG](https://cdn.nlark.com/yuque/0/2023/png/26909528/1692704992335-48ecad66-b367-4a5c-9bfe-623083c294fe.png)选择课程![IMG_0078.PNG](https://cdn.nlark.com/yuque/0/2023/png/26909528/1692704992053-77c7bcb7-49dc-4c8a-847a-a9c37656f162.png)填写信息

（三）进行结算

订单设置

**路径：**点击【结算】 >> 页面下划可设置：整单优惠、优惠券、储值账户抵扣、对内、对外备注等信息

**注：1.**若订单选择设置整单优惠，系统会默认根据不同办理内容的金额进行自动分摊，也可点击【分摊优惠】按钮进行自定义设置

2.设置分摊优惠，需勾选分摊优惠权限点，默认分配给校区管理员、前台、销售角色

![IMG_0992.PNG](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762171582587-4b222709-9a9d-4e78-ae7e-74c477038f4e.png)订单设置![IMG_0993.PNG](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762171594671-3e542fbd-aade-458d-845e-ba95c021d1c0.png)整单优惠设置![截屏 2025-11-03 20.08.36.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1762171741332-30b09026-38a0-444b-82b0-2cdec6a9bd57.png)分摊优惠设置

结算方式

注：1.若在结算有多个办理内容的订单时，实收金额小于应收金额，则可点击【分摊实付】，自定义设置各办理内容的分摊实付金额2.若在若在结算有多个办理内容的订单时，实收金额大于应收金额，则系统会自动将多收款项充值到学员储值账户中3.设置分摊实付，需勾选分摊实付/补费权限点，默认分配给校区管理员、前台、销售角色

面对面收款（推荐）

**路径**：点击【结算】 >> 确认订单，点击提交 >> 选择面对面收款 >> 家长扫码付款，或是机构扫描家长付款码进行收款，这个收款码的有效期是 2 小时

**注：**1. 在线收款需先前往【我的】>>业务设置>>收款设置，开通并绑定收银宝账户，也可以点击查看[收银宝开通流程](https://www.yuque.com/schoolpal-wiki/manual/odxfrv)

2. 在线收款支持机构给家长发送二维码，家长扫码付款，也支持机构直接扫描家长的付款二维码

![IMG_0630.PNG](https://cdn.nlark.com/yuque/0/2024/png/12534096/1723010204487-fec20319-beba-441a-8233-d56ac1125e8b.png)面对面收款![lQDPJwZ8lfuwg9zNCWDNBDiwksOlNMjSI50DjZFY-cAkAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670470637696-78e29eda-6b78-4dfd-aed1-32df9c2bf9b9.jpeg)家长扫码付款

收银宝记账（推荐）

**使用场景：若家长已经提前完成付款，老师可以使用以下方式，将学员课程订单和款项自动匹配**
**还未免费开通收银宝商户？**[**点此查看收银宝开通流程**](https://www.yuque.com/schoolpal-wiki/manual/odxfrv)

根据学员姓名自动关联课程订单与款项

**路径：**选择只记账，点击收银宝 >> 自动关联收银宝账单 >> 点击【查询并关联】 >> 系统将根据学员姓名，自动匹配款项 >> 点击确定，就可以自动完成对账

注：若该学员名下有多笔款项，支持老师根据实际情况选择课程对应款项

![记账收款.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721198812618-68079cbf-f907-4897-9b59-63992725f2af.png)查询并关联![匹配款项.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721198869028-4d9e022f-3b20-49f5-9586-98bce7cd999e.png)选择对应款项![完成对账.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721198837927-74de1cd0-7d62-40bc-80db-850b50bb753a.png)匹配款项![完成对账2.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721198884689-bf990641-54d2-4cd7-a602-79a07cbdb508.png)完成对账

根据支付单号自动关联课程订单与款项

**路径：**选择只记账，点击收银宝 >> 根据家长提供的微信 / 支付宝单号后 8 位进行填写 >> 点击【查询并关联】，自动匹配款项 >> 实现自动对账

![IMG_2380.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721289791510-9699b1f1-dbd6-4b46-80ef-1145045c93d6.png)家长提供后 8 位![单号查询.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721199898045-5d413a66-5bd1-4888-86f4-28da6ac23df1.png)输入数字![匹配款项2.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721199959263-9446def5-73e7-4009-a01d-f19b0208143d.png)匹配款项![完成对账2.PNG](https://cdn.nlark.com/yuque/0/2024/png/45785803/1721199968035-5adb22d0-0e34-4c40-ac2a-76efa23543c7.png)完成对账

其他方式记账

**路径：**选择微信、支付宝等除收银宝外的其他支付方式>>手动输入支付单号、账户和日期>>点击确定，财务进行手动对账

三、引导家校关注

完成**报名续费**后，邀请学员家长关注家校平台入口，老师可以直接出示家校关注二维码，或者发送家校关注短信。
关注家校成功后，家长可以了解学员在校状态、查看学员表现、接收重要通知，帮助机构更轻松地完成基础家校服务，提升家长和学员的认同，增强家校粘性。

App 端

**路径 1：**报名成功 >> 自动弹出引导关注弹窗 >> 点击【现在去】 >> 进行短信 / 二维码邀请关注
