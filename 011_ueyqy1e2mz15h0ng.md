操作步骤

下载外呼通App

**第一步**：安卓手机扫码下载「外呼通App」（iOS、鸿蒙 5.0 及以上系统暂不支持下载）>> 登录时授权「允许拨打电话」、「开启所有通话自动录音」、「授权文件访问权限」、「开启允许后台活动为无限制」

| ![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1772698396315-2409a626-dd49-4ea1-8a52-9cd3bb79bb6e.png)扫码安装外呼通App | ![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1772526853066-1a4e482b-5cce-4ce1-9bc2-f7967e624d2e.png)开启通话和录音相关权限 |
| --- | --- |

**第二步**： 请前往手机 设置 >> 应用管理 >> 找到「外呼通」 >> 耗电管理，将后台活动权限设置为**「完全允许」**注：不同安卓机型对该功能的命名可能略有差异，如「电池优化」「省电策略」等，请根据实际选项操作。

若您使用的是**荣耀手机**：请前往手机 设置 >> 搜索「启动管理」，关闭「自动管理」，选择**「允许后台活动」**

注： 管理者和需要使用外呼的老师，都需要下载「外呼通」App

发起外呼

3.1 确认外呼渠道

如果想要销售老师仅能通过外呼通拨出，可在权限中，取消「本机呼出」权限。

**路径**：内部管理 >> 角色管理 >> 选择需要取消勾选「本机呼出」的角色 >> 点击编辑，搜索「本机呼出」，取消勾选

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1776835855883-5b1d9e9e-f0d0-4362-89c8-d8abf2a877bc.png)

3.2 多场景发起外呼

**路径一**：确保外呼通登录状态，点击校宝的招生中心 >>智能外呼>> 表格视图 /AI 视图（推荐）>> 查看 AI 线索分析概要，快速了解学员信息，点击电话 icon 外呼。外呼过程中，请勿关闭通话页面的录音按钮

**路径二**：确保外呼通登录状态，点击校宝招生中心 >>意向学员>> 表格视图 /AI 视图（推荐）>> 查看 AI 线索分析概要，快速了解学员信息，点击电话 icon 外呼。外呼过程中，请勿关闭通话页面的录音按钮

**路径三**：确保外呼通登录状态，点击校宝教务中心 >> 学员 >> 学员详情 >> 点击电话 icon 外呼

2. 如果外呼发起失败，需要登录外呼通App，查看是否都已同意授权电话、录音相关权限。[点此查看检测步骤](#FPcH8)

常见问题

哪些机型可以下载使用「外呼通App」？

目前仅支持安卓部分主流机型，更多机型适配中。

| **品牌系列** | **适配系统版本详情** |
| --- | --- |
| 小米、红米 | HyperOS 3.0 及以下 |
| 荣耀 | MagicOS 10 |
| vivo | OriginOS 4OriginOS 5OriginOS 6 |
| OPPO | ColorOS 16 |
| 华为 | HarmonyOS 4及以下<br><br>包含HarmonyOS 3.1/4.0及以下、EMUI在内的老系统 |

在校宝后台通过外呼通渠道发起外呼时，提示无法外呼是什么原因？

1. 需要登录外呼通App，查看是否都已同意授权电话、录音相关权限

2. 外呼通要处于登录成功且正常运行状态

3. 耗电管理设置为无限制，[点此查看设置](#sqK3j)

4. 手机不要熄屏

5. 如果是电脑点击外呼，打开外呼通，通话列表显示在屏幕上，不要划走

本机外呼也有录音吗？也会有通话记录转文字、本次通话分析吗？

没有，只有通过外呼通渠道拨打的电话才会有录音以及分析。通过本机呼出的内容受到隐私保护，内容不会上传至系统。

支持同时使用多条外呼渠道吗？

目前同时仅支持选择一条外呼渠道。如果选择使用外呼通，将无法同时选择 TQ 或 EC 外呼渠道。

但机构可以自由切换渠道：

- 多校区：总部管理后台 >> 设置统管 >> 点击授权对接设置 >> 切换外呼通道
- 单校区：点击我的 >> 业务设置 >> 授权对接设置 >> 外呼授权对接 >> 切换外呼通道
