# 配置报表

> 来源: https://www.yuque.com/schoolpal-wiki/manual/yu2xg9vegkxle7sn

场景释义

机构需要查看一段时间内的学员报读课程数据、教师记录课时数据或商品交易收支等数据时，除了系统已经配置好的通用报表，老师可以按自己的需求配置报表，并添加到数据中心，便于后续查看。

手动配置报表

**路径**：PC 端 >> 数据中心 >> 报表管理 >> 汇总表 >> 配置新报表 >> 选择需要的数据明细表（上课记录明细表）>> 选择需要的字段进行配置 >> 生成报表 >> 下载查看数据或保存报表到菜单栏

注：添加字段的说明
支持搜索字段
按左侧维度和指标的分类，分别将字段填入行列规则中
字段的汇总方式支持选择，求和、计数等
支持拖动添加，或点击“+”进行选择
字段默认为系统中的名称，支持重命名
生成报表
默认生成本周的数据，支持编辑修改时间范围
示例
以配置机构【**每个课程】****【到课**或**请假】**的**【学员数据】**和**【记录课时数据】**为例
支持按学员数计数、按课时记录数求和，来统计实际到课的人数和消耗的总课时数
支持将课程加入筛选项，即筛选具体“高三语文课”的到课人数和课消数据

编辑数据中心

**路径**：业务设置 >> 数据中心设置 >> 添加分类，或在已有分类中添加/删除报表

注：新增报表分类上限 20 个，数据分类和分类下面的数据报表均支持拖动排序
自定义报表支持添加到任一分类中，可以拖动排序，但不支持跨分类拖动，即订单交易中的报表不可以拖到账单流水分类中
自定义报表可以添加或移除，系统报表不支持移除，只能拖动排序
有数据中心管理和编辑权限的老师可以操作，默认校区管理员有权限
数据中心编辑设置后，PC 端一级菜单下数据中心中的招生、教务、家校和财务数据报表会同步变动，机构通过权限点的区分，分别给对应老师查看数据

注：总部数据中心的报表管理中，可以配置报表，按校区筛选或分类查看对应的订单、课消、报读、业绩等数据。
总部后台配置的报表不支持添加到数据中心，仅在报表菜单中查看。

常见问题

生成报表后，后续可以二次筛选查看吗？

可以，第一次生成报表，默认筛选本周数据展示，支持编辑修改。

报表支持保存在菜单栏，或添加到数据中心，有对应权限的老师都可以筛选查看数据。

已生成的报表，支持二次编辑配置规则，或复制、删除等操作，默认校区管理员有权限。

订单明细表中可以查看哪些数据

和学员报名订单相关的数据（如订单关联的商品类型、订单应收欠费、订单销售员等），可以以学员、课程、订单类型等为维度查看相关数据。

比如筛选订单经办时间，按销售员查看订单应收实收数据；筛选时间和销售员，按课程订单查看应收、欠费、优惠金额数据；筛选时间，关注应收、欠费、坏账金额，掌握现金流与财务风险。

上课记录明细表可以查看哪些数据

和上课记录相关的数据，如应扣实扣课时、金额，扣费课程账户等，可以以学员、上课日期、上课课程、班级分类等为维度查看相关数据。

学费变动明细表可以查看哪些数据

和学员学费变动相关的数据，如学员转课或退课减少课时数、每日消耗学费、消耗课程账号等，可以以学员、学员变动类型、课程为维度查看相关数据。

报读总表可以查看哪些数据

和学员报读课程相关的数据，如报读课程类型、报读数量和总学费等，可以以学员、课程、收费模式（如按课时、按金额、按时段）为维度查看相关数据。

分班数据明细表可以查看哪些数据

和机构报读学员分班相关的数据，如学员报读课程分班时间、班级上课老师、转入转出班级的学费等，可以以学员、班级信息、班主任为维度查看相关数据。

商品交易明细表可以查看哪些数据

和机构课程、教材商品交易相关的数据，如交易时间、报名课程或购买教材数量、订单实付或优惠金额等，可以以学员、交易时间、交易课程或教程的明细为维度查看相关数据。

业绩明细表可以查看哪些数据

和机构业绩相关的数据，如业绩金额、业绩归属、关联订单等信息，可以以学员、销售员、业绩关联的订单类型为维度查看相关数据。

账单明细表可以查看哪些数据

和机构收支账单明细相关的数据，如收款账户、记账类型（系统同步或手动记账）、关联订单信息（报名、转课等）、账单经办人等数据，可以以收款账户、收支类型、经办人、关联订单等为维度查看相关数据。
