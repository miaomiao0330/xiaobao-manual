# 场地预约付费

> 来源: https://www.yuque.com/schoolpal-wiki/manual/changdiyuyue

一、场景释义

场地预约也是约课的一部分，一些体育类机构没有固定场所进行上课、上课场地、时间、学员等都会随其他因素变化，场地费用也是需要单独支付。为了能保证机构场地成本费用能够高效回收，减少机构与学员沟通时间。故本次新增了场地预约付费功能，帮助机构更好的提升约课效率。

二、操作步骤

（一）创建教室

**场景：**机构可以根据上课时间和教室安排进行创建需要付费的教室，创建后支持修改教室名称及费用

**路径一 ：**校宝 App >> 我的 >> 业务设置 >> 教室设置 >> 点“+”创建教室 >> 输入相关教室名称和场地费用 >> 点击“保存”

**路径二：**校宝 App >> 全部功能 >> 教务管理-约课 >> 点“+”创建约课计划 >> 上课教室 >> 教室管理-点“+”创建教室 >> 输入相关教室名称和场地费用 >> 点击“保存”

说明：① 设置费用后，学员约课时需要支付场地费用，取消约课时，场地费用会原路退回② 课程结束后T+1天，场地费用将自动结算到机构

（二）创建约课计划

**场景：**开课后，机构可以根据老师和教室安排创建预约付费约课计划，开放给报课学员自由选择预约。

**路径：**校宝 App >> 全部功能 >> 教务管理-约课 >> 点“+”创建约课计划 >> 输入相关课程信息 >> 点击“完成”

说明：① 选择上课教室，显示场地费用，自动带入老师创建教室时设置的场地费用，并支持修改② 创建约课计划后，支持修改约课计划详情

（三）家长端预约付费

**场景：**如果机构创建了付费场地的约课计划，在学员预约课程时，支付场地费用后才能预约成功。

**路径：**学员打开“校宝家”小程序 >> 约课 >> 选择多人约课或 1 对 1 约课 >> 选择日期 >> 筛选课程或老师 >> 点击进入约课计划详情 >> 立即预约 >> 支付场地费用

注：学员预约付费成功后，家长和机构收到对应通知

机构取消约课计划

**场景：**机构课程时间安排有变动时，可以取消约课计划（如有有预约学员需要提前告知）。

（1）机构取消：进入约课计划详情，点击右上角取消，取消后，学员支付的场地费用将原路退回

（2）自动取消：到达约课结束时间时，约课人数未达到最少上课人数，约课计划自动取消，学员支付的场地费用将原路退回。

注：约课计划取消后，学员和机构都会收到对应通知
点名或结算后，老师不允许取消约课计划

机构取消学员的约课计划

**场景：**学员时间安排有变动时，如果不方便自己取消约课，老师可以帮学员取消预约，取消后，学员支付的场地费用将原路退回。

**路径：**约课计划详情 >> 点击“取消预约”，即可取消学员的约课计划。

注：机构取消学员约课计划，学员和机构都会收到对应通知
点名或结算后，老师不允许取消学员约课计划

**场景：**学员时间安排有变动，在约课锁定之前可以自主取消预约，取消后，学员支付的场地费将原路退回。

**路径：**“校宝家”小程序 >> 约课 >> 我的约课 >> 已预约日程 >> 取消预约

注：约课锁定后，学员不允许取消约课

三、常见问题

机构可以给学员代约付费场地吗？

不可以

如果将付费预约课程中修改场地费用为 0，可以代学员约课

学员支付场地费用预约成功后可以再预约此课程吗？

不可以

学员支付或机构结算是必须开通收银宝吗？

是的

如果机构未开通收银宝，家长支付费用时会提示“无法购买商品，请联系校区老师处理（原因：微校未设置商户）”

提交订单未支付，可以继续支付吗？

可以

家长可以到校宝家-【我的订单】找到待付款订单在 2 小时之内支付，2 小时后订单将自动取消

家长端可以看到订单信息吗？

可以

学员退费、支付订单、待支付订单可在校宝家校通-【我的订单】中查看

学员支付场地费用预约成功后，费用可以马上结算到机构账户吗？

不可以

要课程结束后T+1天，场地费用将自动结算到机构

费用结算之后学员可以在系统申请退费吗？

不可以

如果结算之后学员需要退费，只能机构走线下退费
