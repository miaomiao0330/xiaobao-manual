# 出入库管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/inventory

一、场景释义

主要为了帮助机构进行精细的教学用品在出入库业务上的管理。

二、操作步骤

PC 端

（一）开启“出入库管理”设置

**路径：**业务设置 >> 内部设置 >> 出入库管理 >> 点击开启“出入库管理”(出入库管理设置**一旦开启不可关闭**)>>机构如果需要自动扣减库存，可以打开“自动扣减库存”开关，老师创建教学用品订单，家长完成付款后，系统自动扣减库存

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1777284772118-040ce8c3-b4ae-4f8f-a0f0-c0de5e917088.png)

（二）货品管理

货品分为统管货品和自建货品，统管货品为总部创建并分发给校区的货品，若总部允许校区自行创建货品，则校区可自行创建，若不允许，则只可应用总部货品

1.创建货品

**路径：**内部管理 >> 出入库管理 >> 货品管理 >> 创建货品

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770184011721-e8a7b46b-7f3f-4e6b-b0a6-6b4c5bae3806.png)

第一步：填写货品名称和货品编号

第二步：开启货品规格，并编辑货品规格，规格名称最多 3 个，每个规格名称下最多添加 30 个属性

第三步：系统会自动根据规格自动组合成不同的商品 SKU

第四步：编辑货品不同 SKU 的成本单价、总部库存以及预警数量（只要货品某个 SKU 的数量低于预警值，就会发送提醒）

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770184156981-bb65ac9e-56fd-4b61-85c1-17ef1641b706.png)

2.出库

当货品报损或其他原因导致库存有减少时，可选择手动操作出库

**路径：**内部管理 >> 出入库管理 >> 货品管理 >> 选择仓库并选择任一货品 >> 出库

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770185917123-893b9eef-3bfb-4464-aa70-d8a49156e410.png)

第一步：选择出库原因

系统默认出库原因有：手动调整出库、报损，若需新增自定义出库原因，可在业务设置 >> 内部设置 >> 出入库管理 >> 出入库原因设置中新增出库原因

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186128057-9e862156-32b8-4fa8-9c82-89646fb97d2e.png)

第二步：选择货品规格

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186166547-5fb63ea6-835c-41b2-b1ca-3cc04c43887a.png)

第三步：填写出库单价、出库数量，完成出库

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186247497-ddb8f413-6c65-44fd-815d-256ff2e99459.png)

3.入库

当货品采购或其他原因导致库存有增加时，可选择手动操作入库

**路径：**内部管理 >> 出入库管理 >> 货品管理 >> 选择仓库并选择任一货品 >> 入库

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186369793-b973186d-5a60-458d-8984-ab27e324b7d5.png)

第一步：选择入库原因

系统默认入库原因有：手动调整入库、采购，若需新增自定义入库原因，可在业务设置 >> 内部设置 >> 出入库管理 >> 出入库原因设置中新增入库原因

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186535232-57af8c08-11b7-401b-9dd5-d1faabd883e0.png)

第二步：选择货品规格

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186847475-472738ab-6436-484b-9c92-26874a090830.png)

第三步：填写成本单价、入库数量，完成入库

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186893923-83d085df-f71b-4f0d-af37-3d453eb276a7.png)

4.编辑货品

**路径：**内部管理 >> 出入库管理 >> 货品管理 >> 选择仓库并选择任一货品 >> 更多 >> 编辑

注：总部统管货品不支持编辑![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770186995298-8ccf9484-8790-4063-b816-9e17c1d4b114.png)

5.删除货品

**路径：**内部管理 >> 出入库管理 >> 货品管理 >> 选择仓库并选择任一货品 >> 更多 >> 删除

注：1.仍有库存的货品不支持删除

2.删除后，该货品的出入库记录仍会保留

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770187408667-8ab3c7b1-ee16-4222-998d-641c685d6a0e.png)

（三）出入库记录

**路径：**内部管理 >> 出入库管理 >> 出入库记录 >> 可查看全部货品的出入库记录，并支持导出

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770187594400-22011b2c-c8e3-4d46-bb29-dcae0aed4b03.png)

三、常见问题

为什么我看不到“出入库管理”功能的入口？

在使用前，需要先进入业务设置，选择出入库管理，点击开启“出入库管理”。

出入库管理可以支持满足记录线下销售商品的场景吗？

可以的，首先需要在货品中添加该商品，在线下销售商品时，选择手动出库，则可以记录线下销售的商品。

在微校卖出的商品，出入库管理功能有相关记录吗？

有的。

1.将需要售卖的商品在“出入库管理”中创建货品。

2.点击“课程” >> 在“商品” >> “教学用品”中点击“创建商品” >> 在“关联货品”中选择对应的货品 >> 设置“报价单” >> 点击上架微校。

该货品则可以通过微校进行售卖，所有的销售信息都会通过出入库管理功能记录下来。
