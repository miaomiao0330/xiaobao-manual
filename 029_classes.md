# 班级

> 来源: https://www.yuque.com/schoolpal-wiki/manual/classes

一、场景释义

班级是机构的基本管理单位，主要管理对象为机构的“正式学员”，报名同一课程的所有正式学员将会组成为一个班级。

二、 操作步骤

App 端

（一）创建班级

**路径：**工作台 >> 班级 >> 创建班级 >> 填写班级信息 >> 选择课模式：课程\组合课程 >> 选择班级对应课程 >> 保存/保存并创建下一个班级

**注：**创建班级时选择对应课程为 A 课程，那么只有报名 A 课程的学员才能进入该班级
创建班级时，可以设置学员和教师分别记录不同课时
创建班级时，若需添加班级属性，需在 PC 端班级属性设置中启用对应班级属性
创建班级支持设置开班时间和结班时间
未到开班时间时，可以排课但无法点名
到结班时间时，当天会给上课老师发消息提醒：班级即将结班

三、常见问题

班级创建好之后可以编辑或者删除吗？

- 班级信息均可以再次编辑，路径：班级详情 >> 右上角“编辑” >> 编辑班级
- 说明：班级在创建和编辑时，班级名称可重名，可通过班级创建时间来区分同名的班级

![lQDPJxzlrYDGuZzNCWDNBDiw87AEi6c1ECIDk-xcHcAQAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670897334978-75c88894-2b8b-46fe-bdde-64ffcd5acdd9.jpeg)班级详情-右上角“编辑”![lQDPJxFIii0cePzNCWDNBDiw0F1Q5IvbvmADk-2M_IAaAA_1080_2400.jpg](https://cdn.nlark.com/yuque/0/2022/jpeg/29343198/1670897388087-5cb02884-c725-4641-9e51-7a900c92111b.jpeg)编辑班级

- 班级创建好之后可以删除，删除后会结班并删除以后课节，“结班”仅意味着班级教学结束，后续不再有任何此班级的课节安排，且“结班”后可以恢复开班，恢复开班后班级将恢复正常状态。
- 删除班级日程路径：教务中心-班级 >> 班级详情 >> 右上角“删除”按钮 >> 删除
- 恢复班级日程路径：教务中心-班级 >> 班级详情 >> 右上角“恢复”按钮 >> 恢复开班

班级已经排好课了，之后还能编辑或删除吗？

班级排好课后，针对「未开始的课节」，点击右上角“…”即可进行编辑或删除；但针对「已结束课节」，点击右上角“…”也可进行编辑或删除。

不小心把其他班级的学员添加进来了，可以将其移出吗？

可以的，您可以将学员批量移出该班级，或者直接调整至另外一个班级。

4. 班级列表中的上课时间为什么有的展示有的不展示？

上课时间会展示已经重复排课的日程，不展示单次排课的日程，单次日程可以在班级详情中查看
