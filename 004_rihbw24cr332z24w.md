# 小红书直连 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/rihbw24cr332z24w

什么是「小红书直连」？
家长给机构小红书私信的手机号、微信号，都能同步到校宝App。校宝App 将自动创建意向学员，方便机构统一归集线索，提升跟进效率。

路径：登录[小红书聚光平台](https://ad.xiaohongshu.com/?redirectTo=%2Faurora%2Fad%2Ftools%2FleadTool%2FleadManage)>> 点击顶部导航栏「工具」、「线索管理」 >> 点击「线索行业工具」 >> 点击「线索推送 API」，查看并复制明文

总部可在校区设置中，允许校区自主添加小红书聚光账号。

四、常见问题

一个小红书号只能绑定一个校区吗？

一个校区可以绑定多个小红书聚光平台的账号；一个小红书聚光账号，只能绑定在一个校区下。

店铺号通过客服接待的线索，也能同步到校宝App 吗？

不能。解决方法：进入千帆App，在客服接待中，将接待工具改成「通过私信联系我」。

小红书的员工号支持吗？

支持。只要是私信线索，小红书聚光平台能收到的线索，都能同步到校宝App。

家长私信留资只有微信号，会同步吗？

会同步，但是因为没有手机号，无法创建意向学员，也不会有自动分配等操作。需要机构去线索列表里查看，手动创建意向学员。
