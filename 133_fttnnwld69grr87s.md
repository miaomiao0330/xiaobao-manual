# 专属小程序（总部）

> 来源: https://www.yuque.com/schoolpal-wiki/manual/fttnnwld69grr87s

一、场景说明

机构可将已经注册并备案的微信小程序作为机构专属小程序，专属小程序用于机构的家校平台，承载线上官网和家校业务办理的能力。总部可统一管理机构的小程序，并将授权的小程序分配给校区进行使用。

授权小程序

**路径：**总部管理后台 >> 设置统管 >> 授权对接设置 >> 微信生态对接 >> 总部设置 >> 专属小程序 >> 点击【已注册小程序，直接授权】 >> 再点击【立即授权】 >> 让对接的**小程序管理员微信**扫描二维码 >> 在微信内确认授权 >> 确认授权成功

注：小程序在总部授权成功后，校区无需再次授权，在总部后台设置所需使用的校区即可直接应用
