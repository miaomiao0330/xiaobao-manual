# 教师课表

场景释义

多校区机构在实际经营过程中，经常遇到以下问题：

1. **教师资源跨校流动：**教师需在不同校区流动授课，但各分校无法掌握教师全局行程。
2. **排课冲突风险高：**分校独立排课时，因无法查看教师在其他校区的课程安排，易导致同一教师在不同分校的授课时间重叠。
3. **规则分散难统一：**节假日、上课时间等规则，需各分校手动单独设置，效率低且易出错，缺乏集中管控能力。

校宝App 总部统管后台，通过总部集中排课、统一设置排课规则、跨校区排课冲突检测等，对排课进行集中管控，提高排课效率！

注：设置节假日时段后，总部可选择同步到校区，校区在创建日程时，可选择过滤节假日，系统即可按照总部设置的节假日时段进行自动过滤

**校区排课规则**

1）查看校区规则详情/设置校区排课规则

**路径：**跨校办理 >> 教师课表 >> 排课规则设置 >> 校区管理 >> 选择校区，查看规则详情/设置

注：1.若设置校区节假日和上课时段同步总部，则校区规则与总部完全同步，且无法修改和新增

2.若设置校内冲突排课设置同步总部，则分校排课发生校内冲突时，会根据总部的配置决定是否可以仍然排课（校区的配置和总部是同步的，且校区无法修改）

3.若设置不允许校区编辑总部创建日程，则校区不可修改或删除总部创建的日程
