# 缺课学员

> 来源: https://www.yuque.com/schoolpal-wiki/manual/wt2tnfnqpn9t7omx

一、场景释义

机构运营过程中，学员持续到校课消才能保证机构健康运营，保证持续产生确认收入将学费落袋。因此机构需要重点关注未按时到课的学员，通过多种渠道批量提醒这批学员需要按时上课。

本次更新新增了缺课学员维度，便于机构通过学员未到校时长，以及请假等缺课情况来筛选缺课学员，通过短信和微信批量提醒学员按时上课，以此来高效解决缺课学员提醒问题，提升机构的课消。

二、操作步骤

（一）自定义配置缺课条件

App 端

配置缺课条件

**路径**：学员管理 >> 缺课学员 >> 点击缺课条件配置 >> 选择学员请假 / 旷课 / 未记录次数，或者距离上次上课天数 >> 通过缺课条件呈现缺课学员数据

配置缺课条件

**路径**：学员管理 >> 缺课学员 >> 点击配置缺课条件 >> 选择学员请假 / 旷课 / 未记录次数，或者距离上次上课天数 >> 通过缺课条件呈现缺课学员数据

注：到课状态最多仅支持查询 30 天内的数据

（二）批量发送缺课提醒

App 端

批量编辑提醒标记

**路径**：学员管理 >> 缺课学员 >> 点击左下方的【批量编辑】 >> 选择需要标记的学员 >> 点击批量修改学员的提醒状态

注：可将退费、情绪学员等标记为无需提醒，避免发送提醒后引起不必要纠纷

![image.png](https://cdn.nlark.com/yuque/0/2023/png/1278685/1700019016817-6f05bde3-518c-4fb5-9a85-be8015bc7bc8.png)![image.png](https://cdn.nlark.com/yuque/0/2023/png/1278685/1700019057176-a5d1d1ad-16d7-4984-b1b3-67a86752e5db.png)![ab500c94f6d2f2af751cb3a067ba023f.png](https://cdn.nlark.com/yuque/0/2023/png/1278685/1700019071490-9dcbe47b-34a5-4ff3-957f-13790c01deb8.png)

筛选缺课状态

**路径**：学员管理 >> 缺课学员 >> 点击右上方的漏斗 >> 选择学员请假 / 旷课 / 未记录次数，或者距离上次上课天数 >> 选择是否需要提醒 >> 筛选出需要发送缺课提醒的学员

通过短信发送缺课提醒

（推荐）

**路径**：缺课学员 >> 点击右下方的【发送缺课提醒】 >> 选择【短信提醒】 >> 选择需要发送缺课提醒的学员 >> 点击发送

通过微信发送缺课提醒

**路径**：缺课学员 >> 点击右下方的【发送缺课提醒】 >> 选择【微信提醒】 >> 选择需要发送缺课提醒的学员 >> 点击发送

注：机构需要引导家长在家校平台关注自己的学员，才可使用该功能。[机构如何启用家校平台？点此查看](https://www.yuque.com/schoolpal-wiki/manual/bld29l)

如家长未在家校平台关注自己的孩子，机构将无法通过微信对未关注家长发送微信推送的缺课提醒。[机构如何提升家校绑定？点此查看](https://www.yuque.com/schoolpal-wiki/wd2t2x/dt0il7ctze0m98rx)
