# 课后任务

> 来源: https://www.yuque.com/schoolpal-wiki/manual/oqikf0

一、场景释义

老师可给学员布置课后任务，用于巩固上课内容，并基于学员的提交的任务了解学员的学习情况和进度。

二、操作步骤

App 端

布置任务

布置单次任务

**路径：**课后任务 >> 点击 「+」 号 >> 填写任务内容 >> 选择仅本次发布 >> 选择需要完成课后任务的学员 >> 选择是否定时、任务截止日期、班级内是否可见 >> 发布

注：1. 任务内容支持上传图片（单张 9M）、视频（单个 500M）、音频（单个 10M）、文件（单个 200M），视频支持在线查看或下载查看

2. 定时发布前，课后任务都可以再次编辑

3.布置课后任务时，可选择与课后作业一致的试卷，在线批改时，可一键将错题收录至该学员的错题本，方便其针对性复习与巩固。

布置定时任务

**路径：**课后任务 >> 点击 「+」 号 >> 填写任务内容 >> 选择设置自动任务 >> 选择需要完成课后任务的学员 >> 选择任务开始结束日期、单次任务时长、班级内是否可见 >> 发布

注：1. 任务内容支持上传图片（单张 9M）、视频（单个 500M）、音频（单个 10M）、文件（单个 200M），视频支持在线查看或下载查看

2. 定时发布前，课后任务都可以再次编辑

3. 超过单次任务时长，学员还未提交，仍可以继续提交

4.布置课后任务时，可选择与课后作业一致的试卷，在线批改时，可一键将错题收录至该学员的错题本，方便其针对性复习与巩固。

AI 批改

**3.1 开启 AI 辅助批改**

**路径 1：**教学中心 >> 作业批改 >> 点击“+”布置课后任务 >> 在布置课后任务时，开启 AI 批改按钮 >> 发布课后任务

**路径 2：**家校服务 >> 课后任务 >> 新建课后任务 >> 在布置课后任务时，开启 AI 批改按钮 >> 发布课后任务

注：1. 必须在布置客户任务时开启 AI 批改按钮，才会启用 AI 批改

1. 建议优先丰富机构的题库，会提高 AI 辅助批改的正确率

![截屏 2026-05-26 21.41.45.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779802954767-d6c3832a-8eca-4c87-8b28-f2eacfd3a4c5.png)布置课后任务![截屏 2026-05-26 21.38.41.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779802776296-c2e2a570-a341-48ce-aefa-3bceff4e5dd8.png)开启 AI 批改

**3.2 AI 辅助批改结果审核**

**路径：**学员提交课后任务后 >> 老师可在课后任务列表找到待批改任务，点击查看任务详情 >> 在 AI 批改已完成列表下，找到对应课后任务 >> 点击 AI 结果审核

![截屏 2026-05-26 21.41.03.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779802976200-a9ee2955-11da-4715-a129-0837a3b6b1c8.png)点击待批改任务![截屏 2026-05-26 21.43.16.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803024626-bc2f2a96-b2f4-47a4-aae8-000f128a5268.png)点击 AI 批改完成学员![截屏 2026-05-26 21.44.07.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803071557-dd1bc819-3819-4a65-b805-11b06be89975.png)点击 AI 结果审核![截屏 2026-05-26 21.45.01.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803116303-ed15a8b0-8ab6-45b3-9423-215a94e37884.png)AI 结果审核

老师需要对 AI 辅助批改的内容核对校验，点击题号或试卷上的题目可切换试题，支持对批改结果纠正，或对识别不准的题目进行再次编辑，可编辑题干、选项、添加解析、选择课纲内容和自定义属性内容

注：1. 标有【新】的题目尚未收录在题库中，需要老师仔细核实

1. AI 辅助批改的课后任务，产生的错题会自动加入到学员错题本中

![截屏 2026-05-26 21.45.01.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803116303-ed15a8b0-8ab6-45b3-9423-215a94e37884.png)AI 结果审核![截屏 2026-05-26 21.46.03.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803178609-2d0ceba4-a85c-4d41-8ab9-c2e0d5a62efc.png)屏幕上划查看详情![截屏 2026-05-26 21.46.42.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779803211420-de4fcc7d-5a82-45b3-b4d8-dd1ea75fcc47.png)编辑试题

若 AI 辅助批改结果不满意，可点击【在线批改】进入手动圈画批改流程，确保全部题目核对无误后，点击右上角的【确认结果并去点评】按钮，进行作业点评

注：老师确认 AI 辅助批改结果后，AI 会自动生成作业点评内容

**3.3 AI 学情分析和学员作业报告**

完成 AI 批改结果审核并点评后，可在课后任务详情页点击【查看详情】，可查看学情分析，包括任务完成情况和题目分析，可查看班级作业的完成情况、班级学生知识点掌握情况、题目分析

注：学情分析仅针对已完成 AI 辅助批改且通过审核的数据进行

圈画批改

**路径**：家校服务 >> 课后任务 >> 选择需要批改的课后任务 >> 在待批下，找到需要批改的课后任务 >> 点击在线批改 >> 进行圈画批改，也可以在错题处或者学员做的好的地方，直接插入录音，学员点击就能听到老师批语 >>布置的作业选择试卷后，可在批改完成后标注错题，将自动进入学员错题本

使用小贴士：1. 家长拍照上传图片顺序混乱，老师批改时支持调整图片顺序

**3.3 设置优秀任务**

**路径**：老师批改、点评完成课后任务后，可以选择设置成精选任务，班级内所有学员都能看到，进行班级内积极的氛围运营

注：如果布置课后任务时，老师设置为不公开，则班级内学员无法查看互相的课后任务；但是老师把课后任务设为精选后，这个任务就是全班可见
