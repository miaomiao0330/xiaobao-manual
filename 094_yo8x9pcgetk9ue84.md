# 财务数据

> 来源: https://www.yuque.com/schoolpal-wiki/manual/yo8x9pcgetk9ue84

场景介绍

帮助机构关注订单收支、账单流水、学费变动等数据，支持筛选时间、选择不同维度查看数据明细，让管理员详细了解机构财务相关数据，掌握现金流水和确认收入，助力机构健康发展。

剩余学费概况

- 数据释义：
- 总剩余学费：尚未产生课消的剩余报读学费金额总和（学员实付学费，不含赠送）
- 总剩余课时：学员报读课程为按课时，且尚未产生课消的剩余课时总和（含赠送课时）
- 总剩余天数：学员报读课程为按时间，且尚未产生课消的剩余天数总和（含赠送时长）
- 总剩余金额：学员报读课程为按金额，且尚未产生课消的剩余金额总和（含赠送金额）
- 操作路径：数据中心>>财务数据>>财务概况>>剩余学费

订单交易概况

- 数据释义
- 订单应收：学员报名订单应收金额（仅包含已完成、待处理订单）
- 订单实收：学员报名订单实际收入金额（仅包含已完成、待处理订单）

账单流水概况

按时间汇总

- 数据释义
- 订单收入：账单中系统同步的订单收入金额
- 订单支出：账单中系统同步的订单支出金额
- 记账收入：账单中手动记账的收入金额
- 记账支出：账单中手动记账的支出金额
- 总收入：订单收入 + 记账收入
- 总支出：订单支出 + 记账支出
- 总结余：总收入 - 总支出
- 操作路径：数据中心>>财务数据>>账单流水>>按时间

按变动类型

- 数据释义：
- 报名学费：学员报名增加的学费。
- 转入学费：学员转入的学费，包含同校或跨校区转课、以及跨校上课转入的学费。
- 返还学费：学费变动类型中所有的返还学费，包含课消退还、撤销结课返还、撤销退课订单等返还的学费。
- 新增学费总计：包含报名学费、转入学费和返还学费总计。
- 课消学费：包含日常课消、导入课消、自动课消、课消补扣、课消欠费清算等所有课消产生的学费。
- 结课学费：包含手动结课、到期结课的学费。
- 转出学费：学员转出的学费，包含同校或跨校区转课、以及跨校上课转出的学费。
- 退费学费：学员退费产生的学费。
- 作废学费：订单作废的学费金额。
- 减少学费总计：变动类型为课消、结课、转出、退费及作废的学费金额总计。课消学费 + 结课学费 + 转出学费 + 退费学费 + 作废学费。
- 确认收入学费：变动类型为全部确认收入类型，包含转/退课手续费。
- 操作路径：数据中心>>财务数据>>学费变动>>按变动类型

常见问题

数据更新时间

- 数据实时更新，App 在页面底部，PC 端在右上角可以看到数据更新具体时间；
- 当调整了数据展示范围时，比如创建一个新的报名订单，会在 5 分钟后更新数据。

不同商品类型订单（课程/教学用品）实收、实退金额的计算方式

当一个订单包含多个商品并产生欠费、补费时，将按商品金额等比例拆分实收/实退

如果作废订单，数据如何统计？

作废订单后，订单收入/支出的数据会在原有的基础上进行更新，覆盖原数据

如果删除上课记录，数据如何统计？

删除上课记录后，确认收入数据会在原有的基础上进行更新，覆盖原数据

如果删除学员，数据如何统计？

删除学员后，历史的收入/支出数据不受影响，无更新
