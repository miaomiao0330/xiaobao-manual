# 待分班学员

> 来源: https://www.yuque.com/schoolpal-wiki/manual/unclassified

一、场景释义

机构在给学员报名后没有给学员分班，老师进入此功能可以看到当前没有分班的学员，点击学员名字可以进入学员详情给学员操作分班

二、操作步骤

App 端

**路径：**首页 >> 学员管理 >> 待分班学员 >> 点击学员档案 >> 去分班
