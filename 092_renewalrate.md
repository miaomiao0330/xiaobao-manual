# 续班率

> 来源: https://www.yuque.com/schoolpal-wiki/manual/renewalrate

一、场景释义

学科机构通常需要按班级维度查看在读学员是否续报下一阶段的课程。比如在寒假班开课前，查看秋季 A 班学员多少已续多少未续，配置统计规则“原班班级、续班班级、时间、课消等”之后，就可以导出相应的续班明细，以此来跟进未续班学员。

二、操作步骤

（一）配置统计规则

**路径：**PC 端数据中心 >> 报表管理 >> 汇总表 >> 续班率 >> 统计规则配置

注：如果选择班级续班级，则选定时间内，学员在对应班级中可以计入应续/已续数据
如果选择班级续课程，则选定时间内，学员报读了目标课程即算作已续
选择班续班时，按老师查看续班率可以选择是否是原班续（即班主任/上课老师是否是原来的老师）

三、常见问题

如果应续条件开启了课消达标要求，学员转过班会有影响吗？

有影响，只有在本班达到了要求的课消数量，才算在应续学员中。

如果已续条件选择报读目标课程，学员转课有影响吗？

仅看所选时间范围内，学员是否有报名目标课程，只要报名了，后期再转课也算在已续学员中。
