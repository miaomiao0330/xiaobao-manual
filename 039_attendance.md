# 上课记录

> 来源: https://www.yuque.com/schoolpal-wiki/manual/attendance

一、场景释义

机构在统计学员课消或教师考核时，老师可以查看学员上课记录明细、老师到课率、班级到课率等各维度统计课消，并根据学员上课记录列表进行数据明细导出，方便机构老师更好的计算和统计课消。

二、操作步骤

App 端

（一）查看上课记录

**路径**：全部功能 >> 上课记录 >> 点击某条记录 >> 查看上课记录详情

PC 端

（一）按学员上课记录

**路径：**教务中心 >> 上课记录

- 进入列表后，系统默认筛选本月已点名的学员数据，老师也可以自定义筛选；还可以按学员维度导出数据并支持查看导出记录，校区数据安全有保障。
- 点击学员姓名进入学员档案，支持查看学员的每一条上课记录。
- 新增拖欠数量字段，上课点名时，当学员点名数量>剩余数量时；系统将自动生成拖欠数量
- 注：默认校长、财务、教务，拥有导出数据权限，其余角色只能查看无法导出。

（二）按日程上课记录

- 进入列表后，系统默认筛选本月已点名的日程数据，同时也支持老师自定义筛选；还可以按日程维度导出上课数据并支持查看导出记录。
- 点击详情，支持查看本节课的上课记录详情，点名详情和点名变更记录一目了然。
- 注：默认校长、财务、教务，拥有导出数据权限，其余角色只能查看无法导出。
