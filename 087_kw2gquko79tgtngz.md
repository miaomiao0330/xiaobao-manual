# 目标管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/kw2gquko79tgtngz

一、场景释义

为了方便机构对整体目标及完成率的统筹管理，同时支持将目标分解到部门及个人，本功能允许学校自定义需要统计的业绩指标，并设定全校目标；随后可根据组织架构按部门、按员工进行目标分解。系统自动汇总计算每个部门及员工的目标完成情况，实时更新全校整体进度。员工还可通过手机端随时查看个人目标完成进度，实现目标透明化管理。

- 校区目标：只能由总部在总部管理后台设置，校区不允许修改
- 部门/个人目标：总部可为校区设置部门/个人目标，为统管目标，校区不可修改；校区可自行设置部门/个人目标

二、操作步骤

PC 端

目标管理

创建目标

**路径**：内部管理 >> 目标管理 >> 创建目标 >> 填写目标名称、年份、统计方式、目标对象、勾选所需要的目标指标

注：1、目标指标分为招生、订单、业绩、课消四类指标，每类指标下含多种指标细项，按需勾选即可为对应指标填写目标值

三、常见问题

各指标的定义说明

| 指标 | 统计逻辑（时间范围：目标中具体年份每个季度或月度） |
| --- | --- |
| 跟进学员数 | 统计员工作为跟进人，跟进时间在时间范围内创建的跟进记录，对学员进行去重统计学员人数 |
| 意向学员数 | 统计员工作为创建人，在时间范围内创建的学员人数 |
| 完成试听数 | 统计员工作为创建人，邀约时间在时间范围内的已试听、已转化的试听记录总数 |
| 订单金额 | 统计员工作为销售员，经办时间在时间范围内的订单明细的实收/实退总金额 |
| 报名续费订单金额 | 统计员工作为销售员，经办时间在时间范围内，订单类型为报名续费的订单明细的实收/实退总金额 |
| 新签订单金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为新签的订单明细的实收/实退总金额 |
| 扩科业绩金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为扩科的订单明细的实收/实退总金额 |
| 续费订单金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为续费的订单明细的实收/实退总金额 |
| 分配业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内的业绩分配记录的业绩总金额 |
| 报名续费业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单类型为报名续费订单的业绩分配记录的业绩总金额 |
| 新签业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为新签的业绩分配记录的业绩总金额 |
| 续费业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为续费的业绩分配记录的业绩总金额 |
| 扩科业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为扩科的业绩分配记录的业绩总金额 |
| 上课消耗课时 | 统计员工作为上课教师，上课时间在时间范围内，上课记录消耗的课时数 |
| 上课消耗学费 | 统计员工作为上课教师，上课时间在时间范围内，上课记录消耗的学费数 |

是否支持给已经设置的目标添加新员工？

支持，在目标列表点击目标名称进入目标详情，点击【添加员工】按钮，即可为这个目标添加新的员工；若需要从目标中删除员工，也可直接找到对应员工进行删除操作
