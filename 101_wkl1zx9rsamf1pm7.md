# 教学云盘

> 来源: https://www.yuque.com/schoolpal-wiki/manual/wkl1zx9rsamf1pm7

一、场景释义

教学云盘将散落在各校区与老师手中的精品课件、试题及讲解视频，统一汇聚为机构的“中央知识库”。

- 当新教师入职，可立即调用成熟教案快速上手
- 跨校区备课时，团队能在线协同打磨，确保教学质量统一
- 老师布置作业或进行课评时，也能直接从库中调取资源，极大提升备课与授课效率

更重要的是，它将每位教师的智慧沉淀为机构可传承、可复用的数字资产，构建起不依赖个人的标准化教学体系，成为支撑机构规模化发展与教学品质长期稳定的核心引擎。

注：教学云盘容量需要扩容，可联系校宝客户经理

二、操作步骤

启用云盘

**路径**：点击左上角总部管理后台 >> 设置统管 >> 教学设置 >> 根据实际使用情况，开启「可用总部云盘」、「启用校区云盘」按钮。支持分校查看总部云盘后，可以让各分校快速获取统一的教学资料，确保教学质量的统一

注：总部管理员可查看并管理全部校区云盘、总部云盘的文件；区域管理员可查看并管理所属部门本级及以下部门校区下的云盘文件

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1780320568946-c79b8d6c-1eb6-467d-bd39-502765018633.png)

总部云盘容量管理

**路径**：点击左上角总部管理后台 >> 机构管理 >> 教学云盘 >> 点击管理 >> 云盘容量管理 >> 各校区设置共享容量、专属容量。云盘容量达到上限时，不会删除已经超出容量的文件，但无法再上传；如需扩容，可联系校宝客户经理

共享容量：多个校区共同使用总部云盘容量

专属容量：校区专属容量，使用不可超出此上限；如果是加盟校机构，可以给每个分校都设置专属容量，保证各分校享用统一的教学资源
