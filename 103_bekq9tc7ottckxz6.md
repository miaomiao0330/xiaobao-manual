# 总部目标管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/bekq9tc7ottckxz6

一、场景释义

为了方便总部对各校区目标值及完成率的统筹管理，同时赋予校区自主分解目标的灵活性，本功能支持总部勾选需要统计的业绩指标，并将目标统一下达至各校区；校区接收目标后，可根据实际情况按部门、按员工进行二级分解。系统自动汇总计算每个部门及员工的目标完成情况，实时更新校区及总部的整体进度。员工还可通过手机端随时查看个人目标完成进度，实现目标透明化管理。

二、操作步骤

创建目标

**路径**：总部管理后台 >> 机构管理 >> 目标管理 >> 选择总部/校区 >> 创建目标

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772611841141-8c6102c0-189b-4bbd-b4dd-23c899d3fd9c.png)

创建校区目标

**路径**：总部管理后台 >> 机构管理 >> 目标管理 >> 总部 >> 创建目标 >> 填写目标名称、年份、统计方式、选择校区和勾选所需要的目标指标

注：1、目标指标分为招生、订单、业绩、课消四类指标，每类指标下含多种指标细项，按需勾选即可为对应指标填写目标值

2、需所属角色拥有总部目标管理相关权限

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772612087352-bfc58e52-d569-42a2-828e-50e0613b4a20.png)创建目标

设置年度目标后，可以将年度目标均摊到每个半年/季度/月份。平分后也可手动再次对目标值进行编辑。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772612812516-dde33f79-893c-4769-9ee4-84c199d263cf.png)填写目标值

创建部门/个人目标

**路径**：总部管理后台 >> 机构管理 >> 目标管理 >> 选择校区 >> 创建目标 >> 填写目标名称、年份、统计方式、目标对象、选择部门、个人和勾选所需要的目标指标

注：1、支持总部老师直接在总部后台给校区创建部门/个人目标，并且总部创建的目标为总部统管目标，校区不允许修改

2、可在设置统管-更多设置中，设置是否允许校区自行设置目标，[点击跳转](https://www.yuque.com/schoolpal-wiki/manual/xkofel5kmfr0wy0i#Fq7XR)

三、常见问题

各指标的定义说明

| 指标 | 统计逻辑（时间范围：目标中具体年份每个季度或月度） |
| --- | --- |
| 跟进学员数 | 统计员工作为跟进人，跟进时间在时间范围内创建的跟进记录，对学员进行去重统计学员人数 |
| 意向学员数 | 统计员工作为创建人，在时间范围内创建的学员人数 |
| 完成试听数 | 统计员工作为创建人，邀约时间在时间范围内的已试听、已转化的试听记录总数 |
| 订单金额 | 统计员工作为销售员，经办时间在时间范围内的订单明细的实收/实退总金额 |
| 报名续费订单金额 | 统计员工作为销售员，经办时间在时间范围内，订单类型为报名续费的订单明细的实收/实退总金额 |
| 新签订单金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为新签的订单明细的实收/实退总金额 |
| 扩科业绩金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为扩科的订单明细的实收/实退总金额 |
| 续费订单金额 | 统计员工作为销售员，经办时间在时间范围内，报读类型为续费的订单明细的实收/实退总金额 |
| 分配业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内的业绩分配记录的业绩总金额 |
| 报名续费业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单类型为报名续费订单的业绩分配记录的业绩总金额 |
| 新签业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为新签的业绩分配记录的业绩总金额 |
| 续费业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为续费的业绩分配记录的业绩总金额 |
| 扩科业绩 | 统计员工作为业绩归属人，业绩经办时间在时间范围内，关联订单和业绩项对应报读类型为扩科的业绩分配记录的业绩总金额 |
| 上课消耗课时 | 统计员工作为上课教师，上课时间在时间范围内，上课记录消耗的课时数 |
| 上课消耗学费 | 统计员工作为上课教师，上课时间在时间范围内，上课记录消耗的学费数 |
