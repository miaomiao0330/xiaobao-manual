# EC 系统对接 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/ec

一、场景释义

接入 EC 外呼服务后，老师可在系统内直接拨号联系学员。

EC 的线索将自动同步至校宝App 并创建意向学员，机构后续可在校宝App 统一跟进，提升效率。

二、操作步骤

EC 后台填写 URL 地址

第一步：EC 后台 >> 应用互联 >> API开放接口 >> 订阅主动推送 >> 粘贴 URL 地址

**URL 地址**：[https://next-api.schoolpal.cn/next-edu-admin/ToB/PC/CallOutCallback/EcSubscribe](https://next-api.schoolpal.cn/next-edu-admin/ToB/PC/CallOutCallback/EcSubscribe)，复制后贴进去即可
