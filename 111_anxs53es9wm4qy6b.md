# 教务设置

> 来源: https://www.yuque.com/schoolpal-wiki/manual/anxs53es9wm4qy6b

场景释义

多校区的课程和班级、课消等规则，总部会进行统一把控，并按照需要分别同步给校区，设置校区的开关权限等。

操作步骤

注：可以批量设置或单独设置
设置了校区同步总部规则后，校区不可以再修改自己的规则
设置了校区同步总部规则后，校区的规则或自定义属性会和总部保持一致，不可修改
允许校区自行创建属性时，校区可以新增自定义课程属性

点名设置

总部设置

**路径**：总部管理后台 >> 设置统管 >> 教务设置 >> 点名设置 >> 总部设置 >> 点名课消顺序设置和课消规则设置

注：机构可以在此设置点名课消顺序、是否支持消超点名，是否开启自动点名，是否限制欠费订单继续课消。

- **设置消超点名开启后**，学员上课点名时支持消超，可以后期补费；如果不开启，不支持消超点名，老师考勤时如果有学员课时不足，会提示移出学员或选择扣除学员其他课程账户的学费。
- **设置自动点名开启后**，未点名的日程会在结束后 30 分钟，系统自动点名并记录课消，此时如果关闭消超点名，有学员课时不足时会自动点名失败。
- **设置限制订单欠费课消后**，学员实付学员消耗完，无法继续课消。
