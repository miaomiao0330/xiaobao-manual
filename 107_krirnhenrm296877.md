# 总部出入库管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/krirnhenrm296877

一、场景释义

在多校区机构的运营体系中，库存与货品的流动直接影响教学保障与运营成本。为确保所有校区货品管理的规范性、可追溯性及成本可控，总部需对全量校区的货品及出入库行为进行统一监管。

出入库管理通过 “总部制定规则、校区执行、流程集中监控” 的模式，实现对所有校区货品流动的标准化管控：

- 对于直营校区，所有出入库流程必须遵循总部制定的规则，使用总部统一的货品目录，确保数据实时同步；
- 对于加盟校区，关键货品（如教材、教具、品牌物料等）的出入库需与总部系统联动；非核心货品可由加盟校自主管理。

通过该机制，总部既能强化货品管控、防范资源流失，又能依托数据整合实现库存优化与采购统筹，支撑多校区业务的稳健扩张。

二、操作步骤

（一）货品管理

总部货品管理

创建并分发货品

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 创建货品 >> 填写货品名称、货品编号，设置货品规格以及成本单价和库存数量 >> 保存并分发 >> 选择校区并确认分发

注：分发货品时，如果校区与总部货品名称/编号重复，会导致分发失败，请修改后再重新分发。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770192440796-6751c309-44bb-40e5-80c2-fad3df3e066f.png)

第一步：填写货品名称和货品编号

第二步：开启货品规格，并编辑货品规格，规格名称最多 3 个，每个规格名称下最多添加 30 个属性

第三步：系统会自动根据规格自动组合成不同的商品 SKU

第四步：编辑货品不同 SKU 的成本单价、总部库存以及预警数量（只要货品某个 SKU 的数量低于预警值，就会发送提醒）

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770102118339-7db0c06c-dd4c-4f64-96ad-696ccaa8407b.png)编辑货品信息

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770102298827-fc69b0a8-7eae-4b2c-9353-a687947d281e.png)选择分发校区

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770102326656-0a80ca6f-727a-4ee8-9943-32e1cd55ab89.png)货品分发成功

分发总部货品

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 选择任意一个货品 >> 点击分发货品 >> 选择校区并确认分发

注：若提示分发失败，可查看分发失败原因，并根据失败原因进行调整后，再重新分发

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770192542370-ed494240-7d34-4096-9f75-becea54aa5fc.png)分发货品

编辑总部货品

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 选择任意一个货品 >> 点击编辑 >> 保存

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770192562122-af4254a7-3848-4a4e-9b7f-1dea5f1326a5.png)编辑货品

删除总部货品

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 选择任意一个货品 >> 点击删除 >> 保存

注：1.总部删除货品后，已分发到校区的货品会同步删除，但出入库记录会保留

2.仍有库存的货品不支持删除，若需删除，需手动清空库存后再删除

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770192620909-dc6d5a3c-8e81-41cc-82c6-4e86bf95e6ad.png)删除货品

查看货品详情

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 点击货品名称查看货品详情

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770203997546-f23ec10b-edce-4a9c-bcdb-ac3b0a399750.png)查看货品详情

**路径：**总部管理后台 >> 跨校办理 >> 出入库管理 >> 货品管理 >> 总部设置 >> 导入/导出货品

三、常见问题

导致货品分发失败的原因有哪些？该如何处理？

| **失败原因** | **处理方法** |
| --- | --- |
| 校区已存在同名货品 | 修改总部货品名称或修改校区已有的货品名称 |
| 校区已存在相同的货品编号 | 修改总部货品编号或修改校区已有的货品编号 |
