# 校宝云打印

> 来源: https://www.yuque.com/schoolpal-wiki/manual/dnr8mr

一、操作步骤

（一）下载客户端

复制此地址：[https://next-fe.schoolpal.cn/next-transfer-fe/xiaobaocloudprinter.html](https://next-fe.schoolpal.cn/next-transfer-fe/xiaobaocloudprinter.html)，请在电脑端打开，即可下载安装

![image.png](https://cdn.nlark.com/yuque/0/2021/png/356079/1623208769697-bcb1783e-05f0-49fa-8eb4-66561fd67fa4.png)

（二）扫码登录客户端

下载完成后，打开校宝手机App，扫描电脑端二维码登录

![image.png](https://cdn.nlark.com/yuque/0/2021/png/356079/1629428306136-3eb1be1c-8338-43b4-859a-e5efef662a03.png)![image.png](https://cdn.nlark.com/yuque/0/2021/png/356079/1623218473568-f88e3c3e-a8cb-4764-b011-9be674de8d1e.png)

（三）添加打印机

注：添加打印机前，请确保电脑与打印机已处于直连状态（Wi-Fi无线连接、USB连接、网线连接等均可）。判断标准：打开电脑任意文件可直接进行打印。

**路径：**工作台 >> 全部功能 >> 校宝云打印 >>添加打印机

（四）打印收据

**路径：**工作台 >> 订单 >> 订单详情>> 查看收据 >>打印收据>> 开始打印

二、常见问题

（一）Mac安装时遇到问题

**路径**：系统偏好设置>>安全性与隐私>>通用>>点按锁按钮>>仍要打开

![image.png](https://cdn.nlark.com/yuque/0/2021/png/356079/1623209146614-11e6823c-adc6-4367-ad8d-97c5f3bb61ca.png)![image.png](https://cdn.nlark.com/yuque/0/2021/png/356079/1623218142399-a6e825ca-783a-4228-97f1-5989446ca4a0.png)

（二）云打印可以远距离使用吗？

可以，云打印对电脑和手机的距离没有要求，只要手机与电脑客户端仍处于绑定状态，且打印机状态正常即可正常使用。

（三）电脑如果断网了还能打印吗？

在断网状态下，此功能将无法使用。
