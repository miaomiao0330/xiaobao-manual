# 教学云盘

> 来源: https://www.yuque.com/schoolpal-wiki/manual/nafy610d2t7t7an9

场景释义

教学云盘将散落在各校区与老师手中的精品课件、试题及讲解视频，统一汇聚为机构的“中央知识库”。

- 当新教师入职，可立即调用成熟教案快速上手
- 跨校区备课时，团队能在线协同打磨，确保教学质量统一
- 老师布置作业或进行课评时，也能直接从库中调取资源，极大提升备课与授课效率

更重要的是，它将每位教师的智慧沉淀为机构可传承、可复用的数字资产，构建起不依赖个人的标准化教学体系，成为支撑机构规模化发展与教学品质长期稳定的核心引擎。

注：教学云盘容量需要扩容，可联系校宝客户经理

云盘文件管理

1.1 文件分类

**我的文件**：每位老师专属文件夹

**共享文件**：校区公开文件夹，所有老师都可以上传文件，并对文件进行管理

**员工文件**：当老师在「我的文件」上传了文件后，会以老师姓名 + 手机号为名称自动创建文件夹；老师离职后，文件夹后会标有「离职」标签；老师修改名字后，文件夹名字也会随之修改

**学员文件**：当在学员详情的「学员文件夹」上传了文件后，会以学员名字自动创建文件夹；学员文件夹最多支持 2 级

注：1. 如总部设置校区不可使用总部云盘，则页面会隐藏总部云盘的入口；如果机构是单校区，则没有总部云盘入口

2. 校区管理员默认可查看和管理所有的文件；老师和教务默认可查看和管理「我的文件」和「共享文件」；教师、销售、顾问、教务默认可查看和管理学员详情页下的「学员文件」

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770280083835-7384f2d8-c897-41b5-b260-3a9d73cc0c50.png)

1.2 上传我的文件

**路径**：教学中心 >> 教学云盘 >> 我的文件 >> 上传文件，也可以直接从桌面拖动上传；通过「我的文件」，管理每位老师自己的教学资料

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770280171433-18fb6d5d-e503-4543-a13e-35b194e8b644.png)

1.3 上传共享文件

**路径**：教学中心 >> 教学云盘 >> 共享文件 >> 上传文件，也可以直接从桌面拖动上传。通过「共享文件」，沉淀机构统一的教学资料；当新教师入职，可立即调用成熟教案快速上手

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770280272538-b0fbca75-8b6f-48ba-9425-9df9174d1501.png)

1.4 员工文件

员工文件仅作为机构统一管理使用，如需上传文件，可以在「我的文件」中进行上传。

当老师在「我的文件」上传了文件后，会以老师姓名 + 手机号为名称自动创建文件夹；老师离职后，文件夹后会标有「离职」标签；老师修改名字后，文件夹名字也会随之修改。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770280318316-9f867a8b-0061-4867-bdca-33252dbe2670.png)

1.5 上传学员文件

**路径**：学员管理 >> 意向/在读/历史学员 >> 选择需要上传文件的学员，在「学员文件」上传文件。老师可以自主选择是否需要在家校端显示，如果开启「家校端」显示的按钮，家长可以在家校平台查看此文件。

上传文件后，教学云盘下的「学员文件」会以学员名字自动创建文件夹。

注：1. 学员文件夹最多支持 2 级

2. 机构云盘容量达到上限时，家长仍可以查看家校端的文件

3. 在家校端开启「学员文件」入口：微校 >> 家校功能入口设置 >> 设置/修改 >> 添加「学员文件」，点击确定
