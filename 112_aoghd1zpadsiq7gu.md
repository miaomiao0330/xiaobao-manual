# 教学设置

> 来源: https://www.yuque.com/schoolpal-wiki/manual/aoghd1zpadsiq7gu

场景释义

教学设置中的教学云盘、题库管理等，由总部统一把控，并按照需要分别同步给校区，设置各校区的编辑/应用权限。

操作步骤

教学云盘

校区设置

**路径**：设置统管 >> 教学设置 >> 教学云盘 >> 根据实际使用情况，开启「可用总部云盘」、「启用校区云盘」按钮。支持分校查看总部云盘后，可以让各分校快速获取统一的教学资料，确保教学质量的统一

注：总部管理员可查看并管理全部校区云盘、总部云盘的文件；区域管理员可查看并管理所属部门本级及以下部门校区下的云盘文件

题库组卷

校区设置

**路径**：设置统管 >> 教学设置 >> 题库组卷 >> 根据实际情况设置是否允许校区编辑总部分发的试题、是否允许校区编辑总部分发的试题、是否允许校区使用 AI 举一反三功能
