# 家校设置

> 来源: https://www.yuque.com/schoolpal-wiki/manual/ovn9ba18gc6m0vcb

场景释义

多校区的机构，可以由总部统一设定机构对学员和家长展示的内容，来规范机构的形象，让机构形象更专业统一。

操作步骤

注：可以批量设置或单独设置校区是否同步总部
设置了校区同步总部后，校区的相关设置会和总部保持一致，不可修改

（六）微校装修设置

注：1.微校装修模板需要选择一个已经开通微校功能校区作为模板，编辑该模板校区的微校后，即可同步给其他校区进行应用，微校内容同步，只会同步如主题、模块顺序等，组件中的内容，如在线商城组件中的课程商品、场地预约组件中的场地等，均需在对应校区内创建后供本校区的微校对应组件引用

2.模板校区微校、机构信息、推广设置均不受总部统管限制，可自由编辑，但是模板校区的微校装修内容修改（如组件、主题修改），会影响其他设置了同步总部微校装修的校区，请谨慎操作

3.装修模板内的字段勾选统管，会把模板校的内容同步覆盖给校区；若不统管则不会同步内容给校区；（如装修模板中的“授课老师”，若勾选，则会把模板校的老师信息同步覆盖给其他校区；反之则不会）

1.总部设置

**路径**：总部管理后台 >> 设置统管 >> 家校设置 >> 微校装修设置 >> 总部设置 >> 设置微校装修模板、装修统管内容、机构信息、推广设置
