# 跨校上课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/bl8fcafrd65thaor

场景释义

有的多校区机构是由总部统一管理课程，会有学员换校区上课、消课的情况。

在确定了课时和上课计划的前提下，可以使用跨校转课，将学员的课程报名账户直接转至新的校区；但遇到学员临时换校区上课时，可进行跨校上课，进行单次消课即可。

三、常见问题

哪些课程可以跨校上课？

按课时、按金额的课程支持跨校上课，按时段的课程无法跨校上课。

跨校上课时，为什么无法选择跨校上课的校区？

如果此学员在 A 校区，现在要去 B 校区上课，老师在操作过程中无法选择到 B 校区，是因为此学员的档案和课程已经关联到 B 校区（关联是指之前已经通过跨校上课，关联档案，而非手动在 B 校区创建同名同联系方式的学员）。可通过以下两种方式查看此学员目前所在校区情况：

- **学员详情**：在 A 校区学员详情 >> 点击跨校课程 >> 查看需要跨校上课的课程 >> 跨校学员详情
- **总部管理后台（仅 PC 端可通过这种方式查看）**：跨校办理 >> 学员管理 >> 查看学员详情

校区 A 的学员在校区 B 消了一次课，课消收入计入校区 A 还是校区 B？

课消收入计入校区 B，学员在哪个校区消课，收入就计入哪个校区。

欠费上课的跨校学员，补费应该补给哪个校区？

学员从校区 A 转到校区 B，在校区 B 产生了欠费，补费也需要补在校区 B。

欠费学员是否支持跨校上课？

欠费学员支持跨校上课。

学员跨校上课，怎么确定是否支持欠费上课？

学员 0 课时跨校上课，只转了一个校区，是否可以欠费上课，取决于当前上课校区的消超规则；

学员 0 课时跨校上课，从校区 A 转到校区 B，再转到校区 C，如果要欠费上课，点名时会去所有的校区找能够扣费的账户，按照创建时间整排序去扣费，如果不够的话，会在本校区产生消超，能不能消超取决于当前上课校区的消超规则配置;

学员如果已经欠了课时，可以从校区 A 转到校区 B，但如果校区 B 不支持消超，则学员无法在校区 B 点名课消。

学员跨到校区 B 上课后，有了点名记录，课堂点评和课后任务怎么查看？

学员家长需要重新在校区 B 绑定自己的孩子，然后切换校区后查看上课记录，点评或课后任务。
