# 数据大屏

> 来源: https://www.yuque.com/schoolpal-wiki/manual/qovriovsncfcnseo

一、场景释义

多校区的机构可以通过总部管理后台查看数据大屏，统一查看各校区的核心数据，实现总部对于校区的统一管理。

二、操作步骤

PC 端

（一）集团数据大屏

**路径：**总部管理后台 >> 数据中心 >> 数据大屏

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770964412131-ee7e7cba-a0e2-4669-8b36-a2dbc10f670a.png)

配置大屏信息

**路径**：点击右上角的配置按钮 >> 配置大屏校区名称、logo

注：机构超管、管理员默认拥有配置大屏基础信息的权限；其他老师如需配置，则需勾选“数据中心”下的“大屏配置”权限

![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733217176759-2c2dfbeb-382b-4ed5-aaf5-bfce3b2ebb6d.png)![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733217197091-bae63a84-4395-4d9b-bb24-e5a9f345487d.png)

查看大屏数据详情

**路径**：点击数据大屏每一块数据，都会跳转至对应的报表明细，或者对应的数据列表，帮助老师定位具体的增长点或者问题点

![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733218565619-327d7105-ad97-4cbc-8cc8-73a0ab98963d.png)
