# 招生数据

> 来源: https://www.yuque.com/schoolpal-wiki/manual/rgg7ll2glvm0yc84

一、场景释义

多校区的机构可以通过总部管理后台，统一查看各校区的招生数据，实现总部对于校区的统一管理。

二、操作步骤

PC 端招生数据

**路径：**总部管理后台 >> 数据中心 >> 招生数据

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770964467734-074998a3-cfcb-4f5c-8d02-b8612c1556c3.png)

App 端

（一）招生 / 教务 / 财务数据

**路径**：点击“我的” >> 总部管理 >> 查看数据报表

注：总部数据为各校区数据的总和
