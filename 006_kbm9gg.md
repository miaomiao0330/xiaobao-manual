# 招生表单

> 来源: https://www.yuque.com/schoolpal-wiki/manual/kbm9gg

一、场景释义

机构会通过地推、转介绍、自然进店等渠道获取到线索，招生表单可以帮助机构快速收集线索信息，线索会自动流入系统，支持员工生成自己的专属推广表单，直观了解员工名下的线索情况，便于绩效考核。

二、操作步骤

App端

（一）创建表单

**路径：**招生表单>>点击右下角“+”按钮创建 >> 填写表单信息 >> 发布

表单支持上传 1 张表单主图、5 张图片和 5 个文本框，并必需要选择渠道。表单支持添加生日、性别、年级等字段，并支持添加设置的自定义字段。

![IMG_7638.PNG](https://cdn.nlark.com/yuque/0/2026/png/12534096/1770049806630-3b0919ca-b73a-4c5a-bc8d-5bec115a39e6.png)右下角创建表单![IMG_7640.PNG](https://cdn.nlark.com/yuque/0/2026/png/12534096/1770049812754-87ef8bc7-a89a-4a90-8ab4-154f3d7b3fea.png)填写表单内容![截屏 2026-02-03 00.39.26.png](https://cdn.nlark.com/yuque/0/2026/png/12534096/1770050404809-a328756d-dbd0-4346-8820-061e44a950b5.png)添加字段

（二）机构推广表单

**路径：**招生表单>>推广>>生成专属表单二维码或链接>>发送给学员

**注：**通过员工的账号分享出去的表单链接或二维码会自动带上员工标识，收集到的线索会自动归属到员工名下。支持用小程序和网页两种方式进行推广：选用微信小程序（推荐），在开启手机号验证的情况下， 无手机验证码按钮，点击联系电话会自动打开微信授权弹窗，以获取手机号；如果关闭手机号验证，则只需手动输入手机号。

（三）家长填写表单

**路径：**点击表单链接/扫描二维码 >> 填写信息 >> 提交

PC端

（一）创建表单

路径：招生中心 >> 招生表单 >> 创建招生表单 >> 输入表单信息、表单介绍等内容 >> 创建

表单支持上传 1 张表单主图、5 张图片和 5 个文本框，并必需要选择渠道。

表单支持添加生日、性别、年级等字段，并支持添加设置的自定义字段。

（二）机构推广表单

路径：招生中心 >> 招生表单 >> 找到表单 >> 推广 >> 下载二维码/海报、复制链接 >> 进行推广

支持用小程序和网页两种方式进行推广。选用微信小程序，在开启手机号验证的情况下， 无手机验证码按钮，点击联系电话会自动打开微信授权弹窗，以获取手机号；如果关闭手机号验证，则只需手动输入手机号，逻辑同选用网页进行推广。

（三）查看招生表单详情

![image.png](https://cdn.nlark.com/yuque/0/2026/png/12534096/1770048655350-59f96dc0-7cfa-4f0c-a5f0-9054e6e7e33f.png)点击表单名称

（四）批量操作表单

路径：招生中心>>招生表单 >> 选中表单 >> 批量删除、转为失效

删除：已推广的表单将无法访问，表单带来的线索仍然保留。转为失效：表单无法提交，列表中仍保留表单，可点击表单名称查看详情。操作需要「转为失效」和「线索导出」权限。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/12534096/1770049162168-0887d1ef-6a49-4609-b031-d02ffd123149.png)批量操作
