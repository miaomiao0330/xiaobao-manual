# 直播课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/classin

场景释义

机构的校宝 App 接入 ClassIn 直播后，老师可以将日程设为直播课，然后把日程直接同步至 ClassIn；学员上课后，根据直播在线时长，校宝系统内自动记录课消。

操作步骤

授权对接

获取密钥

机构已自行购买 ClassIn 服务后，请找 ClassIn 售后索取 SID 和 Secret ，在校宝系统完成绑定。

**路径 1 （多校机构）：**总部管理后台 >> 设置统管 >> 授权对接设置 >> 直播对接 >> 输入 Secret 和 ID >> 完成绑定（总部管理员有权限绑定）

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1767595645408-631798e7-909e-4a2c-b1d5-82b858e496da.png)

**路径 2 （单校机构）：**教学中心 >> 直播课 >> ClassIn 授权 >> 完成绑定

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770288116122-2dfbf88d-7a8a-44e8-b868-46ecd60ec320.png)

回调地址

在校宝系统授权对接后，再将回调地址提供给 ClassIn 售后，即可完成对接。

完成后，学员在 ClassIn 的上课记录会同步到校宝系统。

注：一个 SID 和 secret，只能关联一套校宝系统，如果是多校区机构，超管在总部后台完成授权，所有校区都可以使用，完成日程和课消同步。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1779442762390-6c259674-8fd4-4c14-92c2-1c8a0947c63f.png)

直播类型设置

注：直播类型指的是 ClassIn 直播上课的类型，分 1v1、1v12、大班课三种，均不限制上课人数，会按照同时上麦的学员数区分，老师根据机构的上课类型，选择即可。
1v1 在同一时间仅支持 1 位学员上麦和老师互动；
1v12 在同一时间最多支持 12 位学员上麦和老师互动；
大班课不支持学员上麦，只能文字互动。
直播类型如果需要修改，请在日程开始前 20 分钟完成。

**路径**：业务设置 >> 教务设置 >> 排课设置 >> 直播类型设置 >> 设置日程类型对应的默认直播类型

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770286628509-3f3e6b0f-f428-49ac-9bbb-1002b0be9baf.png)设置直播类型

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770286650472-6397f140-c678-4081-83f0-44bbf9581015.png)编辑修改日程对应的直播类型

直播课点名规则设置

注：规则设置路径
单校区机构：业务设置-教务设置-点名设置（PC 端和 App 都可以）
总部后台规则设置路径：总部后台-设置统管-点名设置（仅 PC 端）
机构开通了直播课后，点名设置中的「自动点名」开关仅控制非直播课日程，「直播自动点名开关」仅控制直播课日程

**路径**：业务设置 >> 教务设置 >> 点名设置 >> 点名课消规则 >> 直播自动点名 >> 开启后编辑到课点名规则 >> 确认

（三）创建直播日程

注：创建日程时，可以选择是否开启直播课，开启直播课后，可以选择是否开启录制；
如果开启录制课程，则直播课结束后学员可以查看回放，否则不能查看回放；
创建好的直播日程，如果在日程开始前 20 分钟增加或移除学员，会自动同步 ClassIn。

创建日程时开启直播课

**路径**：教务中心 >> 班级排课/1 对 1 排课/课表中创建日程 >> 点击开启直播课 >> 选择是否录制 >> 确定即可

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770287041527-9be21454-7ac0-4b7d-ae8c-d234f0eab968.png)创建日程开启直播课

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770287087634-f4989b53-6822-4aa6-91db-98e566f0430a.png)创建好的日程会同步在直播列表

在直播列表直接创建直播日程

**路径**：教学中心 >> 直播课 >> 创建直播日程 >> 选择日程类型创建即可

（三）直播列表查看和点名

日程同步状态

注：同步成功：日程信息已同步至 ClassIn 课程表，教师和学员都可以正常上课；
同步失败：日程信息同步失败或学员同步失败，需要修改后手动重新同步。

**路径**：教学中心 >> 直播课 >> 选择同步成功/同步失败日程

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770287197476-1e36e058-09d1-47e5-ab0e-04721d5fdec6.png)

进入直播间

注：直播详情中的发送上课通知，是再次给日程中的学员通过家校平台发送上课提醒，家长端有链接可以复制进入 ClassIn。
老师点击进入直播间，即可通过浏览器跳转到 ClassIn 客户端，进行授课。

**路径**：教学中心 >> 直播课 >> 选择直播日程点击进入直播间 >> 复制链接进入直播间（跳转 ClassIn 客户端）

![image.png](https://cdn.nlark.com/yuque/0/2026/png/26909528/1770287341062-91b58815-ab3f-47ec-8a3a-0880e8faf552.png)

直播课点名

注：如果开启直播自动点名，直播日程结束后 20 分钟内，系统会根据规则，查看 ClassIn 上学员的在线时长进行点名课消；
如果未开启直播自动点名，老师手动选择当日的直播日程去点名即可；
老师可以在直播统计中，查看学员的直播课到课详情。

三、常见问题

对接后，可以同步哪些信息？

校宝App 创建直播课后，直播课的日程时间、老师、学员可以同步至 ClassIn 的课表，校宝App 的班级信息不会同步

创建的直播日程，显示同步成功，可以再添加学员吗，是否需要再次同步？

在日程开始前，可以添加学员或移出学员，不需要手动点击，修改后会自动再次同步

除了添加学员，还可以修改上课教师、日程时间，都会自动同步

如果是同步失败的日程，按照提示修改后需要手动点击「重新同步」

机构开启了直播课，怎么对接 ClassIn？

可以联系 ClassIn 服务人员，获取 SID 和 secret，在校宝系统授权对接后，再将回调地址提供给 ClassIn，即可完成对接。

对接成功后，机构在校宝系统的排课日程可以自动同步到 ClassIn，老师上完直播课后，学员的上课时长也会同步到校宝系统。

一个 SID 和 secret，只能关联一套校宝系统，如果是多校区机构，超管在总部后台完成授权，所有校区都可以使用，完成日程和课消同步。

机构第一次使用 ClassIn，需要给全体教师和学员先创建账号吗？

不需要，授权对接成功后，在校宝系统创建的直播日程，同步至 ClassIn 时，会自动帮教师和学员创建账号（根据系统内的手机号，默认登录密码 123456）。

如果教师或学员已有 ClassIn 账号，则不会重新创建，用手机号登录即可。

注意：如果机构的教师无大陆手机号，可以登录校宝 App，但是校宝 App 此教师的日程暂不支持同步 ClassIn。

老师在 ClassIn 上课时，查看到自己或学员的昵称和校宝系统不一致？

比如：校宝系统内的学员“平平”（手机号 15389232885），有自己的 ClassIn 账号（15389232885），手机号和系统内的“平平”一致，但昵称为“小林”（或“****2885”），则老师在 ClassIn 平台上课时会看到此学员显示昵称“小林” （或“****2885”），此情况下，可以让学员小林修改自己在 ClassIn 的昵称，和校宝系统保持一致。

哪些老师有权限授权对接 ClassIn？

机构超管；如果是多校区机构，总部管理员也有权限。

哪些老师有权限创建直播课？

默认管理员有权限创建直播课，机构可以为其他角色添加权限。

直播列表的查看和管理：如果老师仅能查看自己的课表，则只能看到自己的直播日程；如果老师有查看全部课表权限，则可以看到全部的直播日程。
