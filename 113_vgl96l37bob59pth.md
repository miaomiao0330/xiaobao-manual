# 财务设置

> 来源: https://www.yuque.com/schoolpal-wiki/manual/vgl96l37bob59pth

一、场景释义

总部财务设置是教培机构财务管理中的关键控制环节，可以确保财务数据的真实性、完整性和可追溯性，为机构经营决策和合规管理提供坚实基础。

按时间段

自动

锁定

**设置结账周期后，每月自动锁定。**

**路径**：选择「按时间段自动锁定」>> 设置或确认结账周期 >> 点击右侧「修改」，任意选择按自然月或非自然月的结账日、锁账日；设置好后，每个周期的账目将会自动锁定

注：1. 在锁定前，确认是否处理所有审批订单；确认是否已核对所有上课记录，锁定后将无法录入结账周期前的未排课点名等；确认是否已完成结账周期内的业绩归属、工资计算等

2. 修改「结账日」后，系统将根据最近一个未锁定的账期，按照最新的规则重新生成结账周期，已锁账的周期不受影响

3. 锁定当前结账周期后，之前的结账周期也将全部锁定，包含断层的时段。例如当前已锁定 6.11-7.10 期间的账目，上一个锁账周期为 5.8-6.7，那 6.8-6.10 期间的账目也将会被锁定

| **举例 1：现在是 6 月 16 日，设置了****每月 12 日自动锁账** |   |   |
| --- | --- | --- |
| **结账周期** | **是否自动锁账** | **解释说明** |
| 5.11-6.10 | 是 | 当月自动锁定时间为 6.12，5.11-6.10 在当月 6.12 结账周期前，因此 6.12 之前的账目将自动锁定 |
| **举例 2：现在是 6 月 16 日，设置了****每月 5 日自动锁账** |   |   |
| **结账周期** | **是否自动锁账** | **解释说明** |
| 5.11-6.10 | 否 | 当月自动锁定时间为 6.5，5.11-6.10 在当月 6.5 结账周期内，因此不会自动锁定 |
