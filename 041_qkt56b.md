# 约课

> 来源: https://www.yuque.com/schoolpal-wiki/manual/qkt56b

一、场景释义

机构创建约课计划或班级开启约课后，学员可以在开放期间自主预约课程，灵活匹配教师时间，提高机构教师和教室的利用率。

二、操作步骤

App 端

（一）约课初始设置

**路径：**校宝App >> 我的 >> 业务设置 >> 约课设置 >> 编辑 >> 保存

- 支持设置约课开放时间、结束时间、锁定时间
- 支持设置家长端是否显示上课人数
- 支持设置约课余额校验规则：最低课时限制、有剩余课时即可约课、允许消超约课
- 支持设置学员约课次数限制
- 支持设置候补约课
- 支持设置请假自动取消约课（含候补）
- 支持设置是否限制学员仅查看自己已报读课程
- 支持设置学校统一上课时段，方便创建约课计划时直接选择

（二）创建约课计划

创建多人或 1v1 约课计划

**场景：**开课后，机构可以根据老师和教室安排创建约课计划，开放给报课的学员自由选择预约。

**路径 1：**校宝App >> 全部功能 >> 教务管理-约课 >> 点“+”创建约课计划 >> 输入相关课程信息 >> 点击“完成”

**路径 2：**校宝App >> 课表 >> 点空白处直接创建日程 >> 选择约课日程 >> 输入相关课程信息 >> 点击“完成”

注：机构可以自由设置重复日程，实现批量创建约课计划；
满班人数：最多上课人数，达到满班人数后，未预约学员无法再预约；编辑预约中或已锁定状态的日程时，满班人数可以调小，但不能小于最少开课人数，及已预约人数；
最少开课人数：如果约课结束时未达到最少开课人数，则约课计划自动取消；
支持选择定向约课班级，选择后，只有班级中的学员可以参与此约课计划。

班级日程开启约课

**场景：**机构可以根据需要，排课时将班级日程直接开放约课，或编辑已有班级日程打开约课开关，方便学员选择预约，有效提升满班率。

**路径一：**校宝App >> 课表 >> 点 “+”创建班级日程 >> 点击开启“支持约课” >> 输入信息后保存即可。

**路径二：**校宝App >> 课表/班级 >> 进入班级课节详情 >> 点击右上角编辑课节 >> 点击开启“支持约课” >> 补充信息后保存即可。

注：班级日程开启约课，保存后不可以取消；
班级结课后，后续约课计划自动取消，如果有学员预约，会发送取消提醒；
添加到班级约课日程中的临时学员，新增电话标记，支持电话联系家长。

（三）学员约课

学员自主约课

**场景：**学员报名了课程后，不确定上课时间，在机构创建好约课计划后，学员可以根据自己的需要选择合适时间进行预约。

注：学员预约成功后，对应教师会收到通知。

（1）多人约课

**路径：**学员打开“校宝家”小程序 >> 约课 >> 选择多人约课 >> 选择日期 >> 筛选课程或老师 >> 点击进入约课计划详情 >> 立即预约

注：进入1 对 1 约课计划详情中，可以当日查看同课程、同老师的所有约课时段，点击可以跳转对应约课详情

（4） 注意事项

- 学员已购买过的课程才有权限预约；若没有购买课程则无法预约。
- 机构在创建课程的时候可将课程同步至微校，若学员没有购买而预约，则会引导至微校购买课程。

候补约课

**场景：**当约课计划已经满员，学员可以候补约课，秉持着先进先出的原则，一有学员取消约课，候补学员即可顶上。

注：系统默认关闭，需要前往业务设置 - 约课设置中打开「候补约课」选项

**学员自行候补路径**：学员打开“校宝家”小程序 >> 约课 >> 选择约课计划 >> 点击“去候补” >> 确认候补

（五）取消约课计划

机构取消约课计划

**场景：**机构课程时间安排有变动时，可以取消约课计划（如有有预约学员需要提前告知）。

①机构取消：进入约课计划详情，点击右上角取消，（如果是重复日程，支持选择删除此节或删除全部）；

②自动取消：到达约课结束时间时，约课人数未达到最少上课人数，约课计划自动取消。

注：约课计划取消后，学员会收到取消通知。

**路径：**约课计划详情 >> 点击“取消预约”，即可取消学员的约课计划。

三、常见问题

约课初始设置有什么？

约课初始设置有时间设置、人数显示设置、时段设置。

（1）约课开放时间

约课计划到达开放时间时，学员可以开始预约；

（2）约课结束时间

约课计划到达结束时间时，学员无法预约，如果此时学员人数未达到最少开课人数，约课计划自动取消；

（3）约课锁定时间

约课计划到达锁定时间时，学员无法取消预约，锁定时间不得小于结束时间；

（4）家长端是否显示上课人数

开启后，多人约课的日程中，上课人数家长可见；

（5）学校统一上课时段

设置学校统一上课时段，方便创建 1 对 1 约课计划时直接选择。

机构可以批量创建约课计划吗？

可以。

机构可以在创建约课计划时，自定义重复规则和课节次数；

创建好的约课计划，可以编辑单次修改或批量修改上课人数、上课教室、授课课时等内容。

机构创建的约课计划，试听学员可以预约吗？

不可以。

机构创建的约课计划，未报名对应课程的学员不可以预约，如果需要试听，老师可以将意向学员添加到日程中。

机构代学员约课，学员如果时间不合适可以取消吗？

可以。

在小程序中，学员打开“我的约课”可以看到机构代约课信息，如果不合适，可以取消。

报名不同课程的学员可以预约同一个约课计划吗？

可以。

机构创建的约课计划，可以选择单个课程或组合课程，选择组合课程时，可以满足报名不同课程的学员预约此日程。

约课计划可以限制报名某个班级的学员才能预约吗？

可以。

机构创建的约课计划，可以选择定向班级，限制某个或某些班级的学员才能预约此日程。

约课日程支持修改吗？能编辑哪些信息，是否影响已预约学员？

可以。

（1）单次约课日程：所有信息均支持修改，已预约学员不受影响，如果修改上课教师，学员会收到通知。

（2）重复约课日程：如果修改上课时间、时长、教师、日期，已预约学员会被取消，并收到通知；如果修改上课人数、授课课时、教室、定向班级、约课说明，则不会影响学员。

班级约课和多人约课的区别是什么？

班级约课有班级固定学员，且日程详情中会展示约课日程所属班级。

已预约班级约课的学员，被加入班级学员列表后，属于约课学员还是班级学员，能取消约课吗？

学员先约课，属于约课学员，可以自主取消约课。

班级约课创建后，有预约学员，老师再添加班级学员时，发现学员总数大于满班人数，怎么办？

不影响，约课日程达到满班人数后，学员不可以再进行预约，但是老师可以添加班级学员。

学员约课成功或取消约课，谁可以收到通知？

教师端，约课成功和取消的通知会发给学员预约课程对应的上课老师；家长端，只要关注了学员或者在学员家庭组中的，会通过公众号推送收到通知。
