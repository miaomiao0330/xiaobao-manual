# 报名优惠 🆕

整单优惠设置

一、场景释义

机构可以为在线商城配置优惠活动，当家长/学员在在线商城下单课程商品进行报名续费时，满足一定条件，就可以选择对应的优惠活动，提高家长/学员报名续费意愿，丰富机构的线上活动类型和体验。

二、操作步骤

（一）配置优惠活动

路径：PC 端财务中心 >> 报名优惠 >> 整单优惠设置 >> 新建整单优惠 >> 输入活动信息 >> 输入规则设置 >> 保存并上架

**优惠生效规则举例说明**当设置相同科目联报 2 个享受优惠时，若学员下单同时购买 3 个课程且满足相同科目条件：**设置满足条件均享受优惠：**3个课程均享受优惠，以此类推（适合“每单购买X个课程以上，满足条件课程均打X折”类型活动）**设置按指定数量享受优惠：**仅对价格最高的2个课程进行优惠，价格最低的课程不进行优惠（适合“每单任选X个课程打X折”类型活动）

（二）编辑和上/下架

编辑优惠活动

路径：PC 端财务中心 >> 报名优惠 >> 整单优惠设置 >> 点击「编辑」按钮

注意：已产生订单、已结束的优惠活动不支持编辑

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1740999564556-cdc3c504-f44d-4984-93ed-3683ea58d7ed.png)

上/下架优惠活动

路径：PC 端财务中心 >> 报名优惠 >> 整单优惠设置 >>点击「上/下架」按钮

状态注释生效中：优惠已启用，微校报名时可使用未生效：优惠尚未生效，达到生效日期后方可使用已结束：优惠已过期，微校报名时无法使用已下架：优惠已下架，微校报名时无法使用

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1740999743097-d775a165-7086-4723-89b8-eb53b897ee0f.png)

（三）使用优惠活动

已上架的优惠活动会展示在微校 - 课程商品详情中

![IMG_8815.PNG](https://cdn.nlark.com/yuque/0/2025/png/43289784/1741004772636-1c4f232e-d0f7-4455-bdc9-7b22f1892646.png)课程商品详情![IMG_8816.PNG](https://cdn.nlark.com/yuque/0/2025/png/43289784/1741004784202-96f9320a-d154-4258-9112-9355351cf568.png)已上架优惠活动列表

将课程商品上架到微校后，家长在在线商城中下单课程商品进行报名续费，满足一定条件，即可选择一个对应的优惠活动，选择后，将自动根据活动信息计算优惠金额。

注意：整单优惠可与优惠券叠加使用，请仔细检查叠加优惠情况，避免优惠过度（优惠券满足金额：总价-优惠活动优惠金额）

（四）统计优惠活动数据

路径：PC 端数据中心 >> 财务数据 >> 订单交易 >> 按整单优惠活动

单课优惠设置

一、场景释义

机构可以在办理报名续费订单时给到学员一定的优惠，此类优惠支持自定义设置：机构统一制定优惠政策和员工自行输入优惠政策。

二、操作步骤

（一）选择优惠模式

路径：校宝 App 端 >> 我的 >> 业务设置 >> 报名优惠设置 >> 选择对应的优惠模式 >> 保存

（二）配置优惠规则

（三）编辑和上架

编辑报名优惠

路径：PC 端财务中心 >> 报名优惠 >> 点击编辑

注意：已产生交易、已结束的报名优惠不支持编辑

上/下架报名优惠

路径：PC 端财务中心 >> 报名优惠 >> 上/下架报名优惠

状态注释生效中：优惠已生效，报名时员工可使用此优惠未生效：优惠未生效，到达生效日期后才可被使用![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1741001359685-5ebf2947-895e-4401-ac89-4bb605a4f448.png)已结束：优惠时效已到期，报名时将不会被使用已下架：优惠已下架，报名时将不会被使用

（四）使用报名优惠

路径：教务中心 >> 报名续费 >> 选择对应优惠

![image.png](https://cdn.nlark.com/yuque/0/2024/png/12534096/1731577583045-0f65c200-c406-4bbd-a835-397c740debb1.png)选择对应优惠
