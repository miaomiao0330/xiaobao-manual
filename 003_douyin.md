抖音和校宝 App 打通，**抖音来客 >> 客资中心的线索能自动同步至校宝App，**自动创建意向学员。同步后的线索可以：

- **在线分配**：筛选抖音直连渠道的线索，机构可以选择分配给指定销售或团队。**销售**直接在**系统内记录沟通**、打标签；**管理者**实时**查看跟进状态与转化进度**
- **分析转化**：分析每周/每月抖音渠道**线索转化率**，基于数据反馈调整投放预算与视频内容方向

注意：此项功能必须要有「抖音来客」后台才能使用，如您还未开通，可点此先开通：![](https://lf3-static.bytednsdoc.com/obj/eden-cn/ljpcylz-lmps/ljhwZthlaukjlkulzlp/pc/audit/favicon.ico)[抖音来客](https://life.douyin.com/p/login)；如需校宝协助开通，可点击链接填写表单，我们将会在第一时间联系您：![](https://img.alicdn.com/imgextra/i4/O1CN013Tjx4s1TS5jYIpZKi_!!6000000002380-2-tps-96-96.png)[抖音来客（抖音开店）申请表](https://alidocs.dingtalk.com/notable/share/form/v01QvjnAAZD6A1vnXo5_i24BhhT_KCAitHE)

绑定抖音来客账号

抖音来客后台授权校宝

**路径**：登录抖音来客后台：[https://life.douyin.com/p/login](https://life.douyin.com/p/login)>> 在店铺管理下，找到「服务应用授权」 >> 在服务商代理下，点击「新增授权」，需要选择的内容如下：

- 所属服务商名称：校宝在线（杭州）科技股份有限公司(校宝在线)
- 授权应用：校宝线索管理
- 授权业务：线索管理解决方案
- 授权功能：线索查询

选择好后，点击提交

校宝后台绑定抖音来客账号

总部统一绑定

1.1 总部后台

**路径**：总部管理后台 >> 设置统管 >> 授权对接设置 >> 选择抖音直连 >> 点击「添加抖音来客账号 ID」按钮 >> 填写账号 ID、账号昵称，选择线索分配方式，点击确定

- **统一分配**：抖音来客账号下的线索统一分配给指定校区。例抖音来客的线索统一分配给校区一，别的校区都不会有这个抖音来客后台的线索
- **按抖音账号 ID 分配**：不同抖音账号下的线索，可以分配给不同的校区，一次性配置多个抖音账号的分配规则。例 A 抖音账号可以分配给校区一、校区二，B 抖音账号可以分配给校区三。一个抖音来客账号下可以关联多个抖音账号
- **手动分配**：从抖音直连同步到总部后台的线索，需要人工分配给指定校区

注：1. 「抖音来客账号 ID」获取方式：登录抖音来客后台：[https://life.douyin.com/p/login](https://life.douyin.com/p/login)>> 点击右上角的 logo，找到账户 ID

2. 「抖音账号 ID」获取方式：登录抖音来客后台：[https://life.douyin.com/p/login](https://life.douyin.com/p/login)>> 点击左侧店铺管理 >> 官方抖音号 >> 找到抖音来客账号关联的抖音账号 ID

3. 总部管理员默认有抖音来客账号配置权限、线索查看分配导出权限；其他老师如果需要操作绑定、查看分配导出线索，则需要分别勾选「抖音直连对接设置」、「查看抖音线索」、「分配线索」、「导出线索」权限

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770884284836-c8312a54-4695-4acb-85bd-23e737af6621.png)

校区各自绑定

2.1 打开允许校区自主添加开关

**路径**：总部管理后台 >> 设置统管 >> 授权对接设置 >> 选择抖音直连 >> 校区设置 >> 打开「允许校区自主添加抖音来客账号 ID」的开关，此方式适用于每个校区都单独运营抖音来客账号的情况

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770884226399-a9c15ca5-0f6b-4242-ae4c-67e581debf65.png)

2.2 校区添加抖音来客账号

**路径**：进入校区管理后台 >> 招生中心 >> 抖音直连 >> 点击右上角的「去设置」，再点击「添加抖音来客账号 ID」按钮 >> 添加抖音来客账号 ID、昵称

注：校区添加抖音来客账号，默认统一分配给本校区；此线索也会同步到总部后台，但会显示已分配的状态

编辑 / 删除抖音来客账号

总部编辑路径：总部管理后台 >> 设置统管 >> 授权对接设置 >> 选择抖音直连 >> 点击编辑，重新选择线索分配方式；或者点击删除和停用，这个抖音来客账号就将无法使用

校区编辑路径：校区管理后台 >> 招生中心 >> 抖音直连 >> 点击右上角的「去设置」，再点击编辑；或者点击删除和停用，这个抖音来客账号就将无法使用

注：1. 校区无法编辑总部添加的抖音来客账号线索分配规则

2. 如校区添加的抖音来客账号，被总部分配给了其他校区；在此校区后台，将无法查看已被分配的抖音来客账号

3. 删除和停用抖音来客账号，不会影响已经创建的线索![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1770892973315-aef6eb21-fa1c-404f-9723-fd755c6c2a1b.png)总部管理后台

常见问题

抖音来客和抖音开店是什么关系？

抖音开店就是在抖音来客后台操作。只有开好店后，才有对应的抖音来客后台。

抖音来客和抖音账号是什么关系？

一个抖音来客后台可以绑定多个抖音账号。

![image.png](https://cdn.nlark.com/yuque/0/2026/png/1278685/1772158860289-e97e5a16-7efa-41f3-bd5e-412a0dd5af6b.png)

一个机构可以绑定多个抖音来客账号吗？

一个机构、一个校区都可以绑定多个抖音来客账号。

一个抖音来客账号可以被不同校区重复绑定吗？

一个抖音来客账号被 A 校区绑定后，无法被 B 校区重复绑定；绑定的账号都会同步到总部后台。

一个抖音来客账号可以被不同机构重复绑定吗？

一个抖音来客账号无法被不同机构重复绑定。

抖音来客后台哪些留资入口的线索可以同步到校宝后台？

抖音来客 >> 客资中心来的线索都支持同步到校宝后台。具体的留资方式可以登录抖音来客后台查看。
