# 学员错题本 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/fzygmoy9ofi9b14p

一、场景释义

老师通过课后任务或拍照试卷给学员收录错题后，则能在校宝 App PC 端快速定位对应学员，查看错题明细、进行错题重练，还可基于已有错题进行 AI 举一反三，智能生成相似题进行巩固训练。

二、操作步骤

PC 端

（一）错题收录

![](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772454079176-686a8f00-7606-470c-a993-8844fb2566d1.png?x-oss-process=image%2Fquality%2Cq_10)[![](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772454079176-686a8f00-7606-470c-a993-8844fb2566d1.png?x-oss-process=image%2Fquality%2Cq_10)AI 错题本老师在线批改作业时标注错题或使用校宝App 拍照上传错题，系统自动加入到该学员的错题本中，并实时同步到家长端校宝App 操作手册](https://www.yuque.com/schoolpal-wiki/manual/bh9iwg26vr09x6i6)

（二）管理学员错题

**路径：**教务中心 >> 学员管理 >> 选中学员 >> 点击错题本 >> 可选择学科学段、筛选题型以及其他筛选条件 >> 可对错题查看错题截图或移除错题

注：老师在学员错题本移除错题后，家校端学员错题本将同步移除

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779958983141-853157f6-a902-416e-9bc5-fbec4ffd8db0.png)

（三）薄弱知识点

**路径：**教务中心 >> 学员管理 >> 选中学员 >> 点击薄弱知识点 >> 可选择学科学段、做题时间 >> 数据总览、可查看学员知识点分析图谱、难易度占比分析

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779963366745-19a83654-0cdf-4671-9f90-e2ad4f053b71.png)

错题重练

**路径：**教务中心 >> 学员管理 >> 选中学员 >> 点击薄弱知识点 >> 选中并点击知识点 >> 错题重练 >> 选择错题加入试题篮 >> 组卷

注：系统会自动根据知识点筛选相关错题

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779963510445-438f73ea-18d0-4391-84b2-128d2cb8e9b0.png)

选择错题

可直接将学员错题加入到试题篮中

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779960525536-4d68ef88-4dfa-443d-a8a8-a2f639f1e33d.png)错题加入试题篮

完成选题并组卷

点击完成选题按钮，进入组卷页面，输入试卷名称、学科学段等基础信息，并选择【是否需要添加个性题】

注：开启后，在下载试卷或布置作业时，会从学员错题本中自动选择对应试题加入到试卷中

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779961339668-f967ceed-6238-4373-9aad-19f6bbb24eae.png)

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1779961965338-798dd128-4329-4be8-a2da-5303d3e0a058.png)

1.3. 下载试卷

完成组卷后，即可去下载试卷，下载时，可选择仅通用题、仅个性题或通用题+个性题三种试卷形式

注：选择多名学员时，个性题将自动从学员错题本中选取后插入到试卷中，也可手动为每个学员从题库或者错题本中选择个性题

AI 举一反三

错题举一反三

**路径：**学员详情 >> 薄弱知识点 >> 错题重练 >> 对错题进行 AI 举一反三，将举一反三推荐的相似题加入到试题篮中

注：AI 举一反三每次为你推荐 1 道相似题，每道错题最多推荐 3 道相似题
