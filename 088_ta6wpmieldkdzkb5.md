# 个人数据中心 🆕

> 来源: https://www.yuque.com/schoolpal-wiki/manual/ta6wpmieldkdzkb5

一、场景介绍

通过业绩数据、授课数据、跟进数据，让销售老师、上课老师可以直观看到个人业绩指标情况，做好过程管控，对机构业务结果负责。

二、场景释义

App 端

1.1 业绩数据报表

**路径**：点击“我的”>>老师根据筛选业绩分配时间，查看不同时间段的个人业绩情况

注：可在个人数据权限下，给老师配置“可见我的业绩”权限，每位老师就可以查看关于自己的业绩

![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733231798197-a9495913-0c10-49a9-b85f-c85047167c62.png)![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733231832271-56ef8731-9f2e-43ec-8364-13a944f68452.png)

1.2 老师授课数据

**路径**：点击“我的”>>上课老师可以查看个人名下的出勤率、授课课时、消耗课时，也可以通过筛选时间，知道不同时间段的课时情况

注：1. 可在个人数据权限下，给老师配置“可见授课数据”权限，每位老师就可以查看关于自己的课时情况

2. 点击数据旁的问号标志，可以查看每个数据的统计维度

3. 当无上课记录中的“查看课时汇总学费消耗”权限时，“点名消耗学费”指标不进行展示

![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733232207406-162915ef-997e-4d1c-8be5-1377677402a1.png)![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733232326151-67e32066-d5b9-4c2b-b253-295532a7f4f2.png)![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733232628952-73e1959b-0ab0-4e93-9f33-65ed50baba22.png)

1.3 销售跟进数据

**路径**：点击“我的”>>销售老师可以查看个人名下的线索跟进情况，也可以通过筛选时间，知道不同时间段的线索数据

注：可在个人数据权限下，给老师配置“可见跟进数据”权限，每位老师就可以查看关于自己的跟进数据

PC 端

2.1 数据简报

**路径**：首页>>数据简报

**数据统计维度：****本周总收入**：等同于财务数据>>财务概况>>按时间>>总收入
**本周总支出**：等同于财务数据>>财务概况>>按时间>>总支出
**本周确认收入学费**：等同于财务数据>>财务概况>>按时间>>确认收入学费
**本周学员出勤**：等同于教务数据>>出勤概况>>按时间汇总

![image.png](https://cdn.nlark.com/yuque/0/2024/png/1278685/1733309626136-a3db721d-bc60-4a46-9338-9c3caf92602d.png)

2.2 数据概况

订单收入概况

- **现金总收入**：等同于财务数据>>订单概况>>按时间>>订单实收
- **收入类型**：等同于财务数据>>订单概况>>按商品类型>>订单实收

课消概况

- **实际课消**：等同于教务数据>>课消概况>>按时间>>消耗课时数量
- **计划课消**：等同于教务数据>>课消概况>.按时间>>计划课时课消

剩余学费概况

- 等同于财务概况>>剩余学费

学员出勤概况

- 等同于教务概况>>出勤概况>>按时间

学员人数概况

- **在读学员**：等同于在读学员列表数量
- **意向学员**：等同于招生数据>>招生概况>>按时间>>当前意向学员数
- **历史学员**：等同于历史学员列表数量
