# 机构管理（员工、校区）

> 来源: https://www.yuque.com/schoolpal-wiki/manual/qlg6fk81xfod0ete

一、场景释义

多校区的机构可以通过总部管理后台，统一管理各校区的员工入离职、员工的角色权限问题，实现总部对于校区的统一管理。

二、操作步骤

App 端

（一）员工管理

新建员工

**路径：**登录校宝 App >> 点击我的，进入总部管理 >> 点击员工管理 >> 点击右下角的【+】创建员工，填写员工姓名、手机号、所属组织、总部角色、任职校区与角色、员工状态

注：1. 员工归属某一个组织后，可通过选择【总部角色】来决定该员工在本组织内的权限范围

2. 【任职校区和角色】不会影响员工在组织下的权限范围

（三）部门架构管理

多校区一般的组织机构由统一管理部门，和各校区的人员组成；业务类型一般有直营、加盟的模式。

总部组织架构可以清晰划分机构的职能部门，同时通过【总部角色】决定某个职能部门老师的权限。

例：A 老师在杭州总部，负责机构所有的跨校转课业务，就可以在杭州总部下设教务部，将老师的【所属组织】勾选到杭州总部下的教务部，然后在【总部角色】中，选择跨校转课的权限，A 老师就可以管理杭州总部下所有的跨校转课业务。如该老师同时负责机构校区一的教学管理，则可在【任职校区和角色】中选择校区一，并设置对应的管理权限。

新建部门架构

**路径**：机构管理 >> 部门架构 >> 新增一级部门 >> 点击确定

注：完成一级部门创建后，支持在一级部门架构下创建二级、三级部门架构

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1752746862199-d288500c-8594-48ac-9730-2824ca6896ab.png)

管理组织架构

**路径**：点击组织架构旁的“+”、“...”>>选择新增、编辑或删除该组织架构

注：组织架构下没有校区和员工才可删除

（四）管理员工

新建员工

**路径：**机构管理 >> 部门架构 >> 新建员工 >> 填写员工姓名、手机号、所属组织、总部角色、任职校区与角色、员工状态

注：1. 员工归属某一个组织后，可通过选择【总部角色】来决定该员工在本组织内的权限范围

2. 【任职校区和角色】不会影响员工在组织下的权限范围

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1752746998712-15b06441-7307-44d1-a372-51851a92ecf4.png)

编辑员工

**路径：**机构管理 >> 部门架构 >> 选择需要编辑的任意一位员工 >> 修改头像、姓名、所属组织、总部角色、账号状态等

注：员工手机号激活后，管理无法进行更改；如要修改，建议员工自己登录 App 点击“我的 - 编辑信息”进行修改

![image.png](https://cdn.nlark.com/yuque/0/2025/png/12534096/1736248961981-345474b6-38b1-418f-996a-bb64fa1fccee.png)

离职员工

**路径：**部门架构 >> 选择需要离职的任意一位员工 >> 点击确认知晓风险>>操作离职

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1752747064278-06469b17-12bd-4f0e-912a-092b783d6263.png)

批量操作

路径：部门架构 >> 选择员工 >> 选择对应的批量操作

批量操作包括：批量修改所属总部部门、修改总部角色、修改所属校区部门/角色、离职、复职

![image.png](https://cdn.nlark.com/yuque/0/2025/png/43289784/1752747101723-58a99997-3710-4378-925a-b82c73190d04.png)
