# 欠费学员

> 来源: https://www.yuque.com/schoolpal-wiki/manual/nulmgu

一、场景释义

有时家长在报名时可能无法一次性缴清全部学费，机构为了留住生源，会先让家长支付订单的部分金额，学员便可以正常开始上课，家长只需在一定时间段内将剩余学费补齐即可。以上场景就是我们所说的欠费报名，校宝App提供了对应功能来满足机构需求，支持家长**一次性或多次补缴订单欠款**，**也可以在按课时模式和按时段模式下仅清算或扣课时清算。**

二、操作步骤

App 端

（一）欠费报名

**路径：**报名续费 >> 填写报名信息 >> 确认并提交订单 >> 填写实收金额 >> 点击确定

（三）课消欠费

- 学校在允许消超点名的情况下，学员课时不足时仍然进行了上课点名，产生课消欠费

（四）补缴课消欠费

注：按课时模式支持仅清算和扣课时清算；按时段模式支持仅清算
如果选择仅清算，会直接清空拖欠课时，清算后不可撤回
如果选择抵扣并清算，会按照当前已有课时数自动选择可抵扣的拖欠课时数，抵扣后会计入确认收入

三、常见问题

欠费订单可以多次补缴吗？

可以的，一笔订单支持多次补缴，每一次的支付记录均会体现在订单中。
