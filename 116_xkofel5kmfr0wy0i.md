# 更多设置

> 来源: https://www.yuque.com/schoolpal-wiki/manual/xkofel5kmfr0wy0i

出入库管理

一、场景释义

主要为了帮助机构进行精细的教学用品在出入库业务上的管理。

二、操作步骤

总部设置

**路径**：总部管理后台 >> 设置统管 >> 更多设置 >> 出入库管理 >> 总部设置 >> 新增或编辑出入库原因、开启库存自动同步

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770183091831-3df285e6-e1fb-4f29-9097-ffc4ff825c64.png)

**出入库原因设置**

系统提供 4 个默认入库原因和 5 个默认出库原因，可新增自定义出入库原因，出入库原因分别最多 20 个

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770183207655-9c4cde4e-e671-468d-b7e7-0481275e3c63.png)

**库存自动同步**

开启后，系统将根据教学用品的订单类型自动同步库存：销售订单实时扣减库存并出库，退费订单实时增加库存并入库

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770183283996-64bd04f7-2520-45a8-9704-27908adbfa95.png)

校区设置

**路径**：总部管理后台 >> 设置统管 >> 更多设置 >> 出入库管理 >> 校区设置 >> 设置是否同步总部出入库设置以及是否允许校区自行创建出入库原因

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1770183403059-5b36d7d9-08be-4299-b279-b59f4c48d419.png)

目标管理

一、场景释义

设置是否允许校区自行制定部门和员工目标

二、操作步骤

**路径**：总部管理后台 >> 设置统管 >> 更多设置 >> 目标管理 >> 校区设置 >> 设置是否允许校区自行设置后目标

![image.png](https://cdn.nlark.com/yuque/0/2026/png/43289784/1772619777613-2424d70e-c93e-4b59-b5dc-8bc9a526e61b.png)
