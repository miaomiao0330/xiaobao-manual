# 报表管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/hzea6ma0nnllrg4o

一、场景释义

多校区的机构可以通过总部管理后台，统一查看和导出各校区的报名订单数据、上课记录明细等数据，实现总部对于校区的统一管理。

二、操作步骤（仅 PC 端）

（一）PC 端登录

路径：复制网址并用浏览器打开[https://pc.schoolpal.cn/](https://pc.schoolpal.cn/pc/login)>> 校宝App 右上角扫码按钮扫描 PC 端二维码或账号密码登录

（二）报表配置及查询、下载

订单明细表

**路径：**总部管理后台 >> 数据中心 >> 报表管理 >> 订单明细表 >> 选定时间进行配置 >> 生成报表 >> 生成后自动下载，或在报表生成记录中查看、下载

注：暂不支持查看跨校转课或者跨校上课的订单明细。
