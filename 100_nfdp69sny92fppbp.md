# 角色管理

> 来源: https://www.yuque.com/schoolpal-wiki/manual/nfdp69sny92fppbp

一、场景释义

多校区的机构可以通过总部管理后台，统一管理各校区的员工的角色权限问题，实现总部对于校区的统一管理。

二、操作步骤

（一）总部角色

注：1.总部角色用于给员工分配总部管理权限，如办理跨校转课等2.系统默认提供 3 个总部角色，且默认总部角色不可编辑

新建角色

**路径：**总部管理后台 >> 机构管理 >> 角色管理 >> 总部角色 >> 新建角色 >> 填写角色名称、角色描述、选择权限（支持引用已创建的角色权限） >> 保存

（二）校区角色

注：1.校区角色用于给员工分配校区的管理权限2.系统默认给每个校区提供 7 角色，默认角色会分发给全部校区，不可手动分发且不可删除3.总部可新建角色后选择分发给指定校区，并可选择校区是否同步总部角色，若选择同步，则校区不可编辑该角色，且总部修改角色，已分发的校区角色自动同步更新4.总部可设置是否允许校区自行创建角色，若允许，则校区可创建，若不允许，则校区只能使用总部分发的角色，不允许自行创建

三、常见问题

1. **系统默认角色是否可以编辑修改？**

默认总部角色不允许编辑，默认校区角色允许编辑

1. **为什么校区的角色统管标志有彩色有黑白，有什么区别？**

带统管标志的角色都是由总部分发的角色，彩色统管标志为开启了自动同步总部的角色，黑色统管标志为未开启同步总部的角色
